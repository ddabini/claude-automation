닉네임 설정

닉네임을 입력해 주세요.

링크로 등록

Firebase Storage URL 또는 외부 URL을 입력하면 iframe으로 프리뷰합니다.

프로젝트명을 입력해 주세요.
올바른 URL을 입력해 주세요.
Firebase Storage 공개 URL, Firebase Hosting URL, 또는 iframe으로 표시 가능한 외부 URL을 입력하세요.

Figma로 보내기

현재 디자인을 Figma 캔버스에 캡처하여 보냅니다.
아래 순서대로 진행해 주세요.

① Figma 데스크탑 앱을 열고 캔버스를 준비하세요.
② 아래 "캡처 스크립트 복사" 버튼을 클릭하세요.
③ Figma 플러그인 콘솔에 붙여넣고 실행하세요.
④ 현재 디자인 URL이 자동으로 캡처됩니다.
