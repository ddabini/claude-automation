## 코드 구조 개요

### pikbox.html (약 950줄)

단일 HTML 파일에 CSS와 JS가 인라인으로 포함된 완전한 웹 앱.

#### HTML 구조
- `<header>`: 앱 제목, 에이전트 내보내기, 설정 버튼
- `<aside.sidebar>`: 카테고리 목록, 새 카테고리 버튼
- `<div.main>`: 액션 바 + 벌크 액션 바 + 이미지 그리드/빈 상태
- 모달 5종: 설정, 카테고리, Run, 상세, 이동
- 우클릭 컨텍스트 메뉴, 토스트 컨테이너

#### CSS (약 350줄)
- CSS 변수로 디자인 시스템 정의
- 사이드바 + 메인 2단 레이아웃
- CSS Grid (auto-fill)로 반응형 이미지 그리드
- 모바일/태블릿 브레이크포인트 (480px, 768px)

#### JavaScript (약 550줄)
- **설정 관리**: localStorage에 API 키 5개 저장/불러오기
- **Supabase 통신**: sbFetch() 헬퍼로 REST API 호출
- **카테고리 CRUD**: loadCategories(), saveCategory(), deleteCategory()
- **이미지 업로드**: Storage POST → 공개 URL 생성 → DB INSERT
- **이미지 표시**: renderImages() — 필터/정렬 적용, 카드 HTML 생성
- **멀티선택**: toggleMultiSelect(), bulkMove(), bulkDelete()
- **Run 기능**: doSearch() → searchSerperImages() → 결과 모달
- **에이전트 내보내기**: exportForAgent() — MD 파일 생성 + 다운로드

## 외부 의존성

| 의존성 | 유형 | 용도 |
|--------|------|------|
| Noto Sans KR | Google Fonts CDN | 본문 폰트 |
| Supabase REST API | 런타임 API | 데이터 저장/조회 |
| Supabase Storage | 런타임 API | 이미지 파일 저장 |
| Gemini 1.5 Flash | 런타임 API | 이미지 분석 |
| Serper.dev | 런타임 API | 이미지 검색 (Google 이미지 래퍼) |

## 보안 고려사항

- API 키는 localStorage에만 저장 (코드 하드코딩 없음)
- HTML 이스케이프 함수(escHtml)로 XSS 방지
- Supabase RLS 활성화 (개인 도구이므로 anon 전체 허용)
- 이미지 업로드 시 파일명 랜덤화로 충돌 방지
