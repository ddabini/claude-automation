## 4. 컴포넌트 상세

### 4.1 이미지 카드
- 정사각형 썸네일 (aspect-ratio: 1)
- 좌상단: 체크박스 (멀티선택 모드 시)
- 우상단: 플랫폼 뱃지 (색상별 구분)
- 하단: 제목 + 키워드
- 호버: 액션 버튼 오버레이 (상세/링크/삭제)
- 선택 상태: 인디고 보더 + 그림자

### 4.2 플랫폼 뱃지 색상
| 플랫폼 | 색상 |
|--------|------|
| uploaded | 인디고 (#6366f1) |
| pinterest | 빨강 (#e60023) |
| behance | 파랑 (#053eff) |
| instagram | 핑크 (#e1306c) |
| web | 검정 (60% 투명) |

### 4.3 모달 종류
1. **설정 모달**: API 키 5개 입력
2. **카테고리 모달**: 이름/설명/색상 입력
3. **Run 모달**: 진행→키워드→검색결과→저장
4. **상세 모달**: 이미지 미리보기 + 제목/메모/키워드 편집
5. **이동 모달**: 대상 카테고리 선택

## 5. 사용자 플로우

### 최초 사용
1. 앱 열기 → 설정 모달 자동 표시
2. API 키 5개 입력 → 저장
3. 카테고리 생성 → 이미지 업로드 시작

### 일반 사용
1. 카테고리 선택 → 이미지 드래그앤드롭 업로드
2. 업로드된 이미지 선택 → Run 클릭
3. AI가 이미지 분석 → 유사 레퍼런스 검색
4. 검색 결과에서 마음에 드는 이미지 선택 → 저장
5. 필요 시 멀티선택으로 이미지 정리 (이동/삭제)

### 에이전트 연동
1. 원하는 카테고리 선택
2. [에이전트 내보내기] 클릭
3. MD 파일 다운로드
4. 프로젝트 레퍼런스 폴더에 저장
