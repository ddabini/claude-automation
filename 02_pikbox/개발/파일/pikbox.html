Firebase 클라우드 저장 활성

모든 카테고리·이미지 데이터가 Firebase에 영구 저장됩니다.

🔍 Run 기능 — 광고 크롤링 & 디자인 레퍼런스 검색

Instagram·메타 광고 라이브러리·Pinterest·Behance·TikTok·YouTube·네이버 등에서
SNS 광고 소재, 카드뉴스, UI 디자인 레퍼런스를 업종별·유형별로 수집합니다.
Serper API Key를 입력하면 전체 플랫폼 검색이 활성화됩니다. (월 2,500회 무료)

serper.dev → 회원가입(Google 로그인) → Dashboard → API Key 복사
* 월 2,500회 무료 · Google 이미지 검색 결과를 JSON으로 반환
pixabay.com/api/docs → 회원가입 → API Key 복사
* 한 번에 200개 · 시간당 5,000회 무료 · 고품질 이미지/일러스트/벡터

아래는 선택사항입니다. 입력하면 추가 이미지 소스가 활성화됩니다.

unsplash.com/developers → New Application → Access Key
pexels.com/api 가입 후 → Your API Key

📘 Meta Marketing API

Meta 광고 소재별 노출·클릭·전환 성과를 실시간으로 불러옵니다.
Long-Lived Token은 Graph API Explorer에서 발급 가능합니다.

만료 60일 · 7일 전 자동 경고 알림 · 앱 권한: ads_read, read_insights 필요
Meta 광고 관리자 → 계정 설정 → 광고 계정 ID (act_ 포함 입력)

🔵 Google Ads API (Apps Script 프록시)

Google Ads API는 CORS 제한으로 브라우저에서 직접 호출 불가합니다.
Google Apps Script 프록시를 통해 데이터를 가져옵니다.
script.google.com → 새 프로젝트 → 프록시 코드 배포 후 URL 입력

Google 광고 → 계정 설정 상단의 고객 ID (xxx-xxx-xxxx 형식)
