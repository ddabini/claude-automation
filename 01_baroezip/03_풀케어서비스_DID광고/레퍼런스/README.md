## 공통 트렌드 요약

### 컬러
- 다크 배경(딥 네이비 #0D1B2A ~ 차콜 #1A2E4A)이 "프리미엄 + 권위" 표현에 가장 효과적
- 골드(#C9A96E) 액센트로 CTA, 핵심 수치, 포인트 라인 처리
- 2025 트렌드: 다크+포인트 컬러의 볼드 대비가 주류

### 타이포그래피
- "빅타이포" 트렌드 강세 — 헤드라인이 화면 30-50%를 차지
- ExtraBold/Black 타이틀 + Regular 서브텍스트로 드라마틱 계층 구조
- 한글: Paperlogy ExtraBold (타이틀) + Pretendard (본문)

### 레이아웃
- 세로형: 상단 후킹 → 중단 서비스 내용 → 하단 CTA의 3분할 구조
- 카드형 2×2 그리드로 서비스 항목 시각화
- 골드 포인트라인으로 섹션 구분 + 시선 흐름 가이드

### 비주얼 장치
- 글로우 효과, 그라디언트 오버레이, 투명 카드 레이어
- 숫자 배지로 신뢰 수치 강조
