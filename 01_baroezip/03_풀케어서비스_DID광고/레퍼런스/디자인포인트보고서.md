## 2. 레퍼런스별 디자인 포인트 분석

### 레퍼런스 ①: Dark Background + Gold/White Real Estate Ad
**출처**: PosterMyWall 부동산 광고 템플릿 컬렉션

**디자인 포인트**:
- **배경**: 딥 네이비(#0A1628) 또는 차콜 블랙(#1A1A2E)
- **메인 컬러**: 골드(#C9A96E ~ #D4AF37) — 고급스러움과 신뢰 표현
- **서브 컬러**: 크림 화이트(#F5F0E8)
- **레이아웃**: 상단 히어로 텍스트(대형) → 중단 서비스 포인트 → 하단 CTA
- **후킹**: 숫자 강조("4명의 전문가", "등기까지"), 대형 쿼트 마크

**적용 포인트**: 다크 배경이 광고판/DID에서 시선을 확실히 잡음. 골드 액센트는 프리미엄 인식 강화.

---

### 레퍼런스 ②: 세로형 DID 광고 디자인 원칙 (Screenly)
**출처**: Screenly Vertical Digital Signage Guide 2025

**디자인 포인트**:
- **5x5 룰**: 한 화면당 5줄 이내, 줄당 5단어 이내
- **폰트 크기**: 시청 거리 고려 → 타이틀 최소 120px 이상(2160px 기준)
- **계층 구조**: 타이틀(80%) → 서브타이틀(50%) → 본문(30%) 비율
- **여백**: 상하 여백을 충분히 확보해 "숨통 트이는 레이아웃"
- **비주얼**: 고품질 배경 이미지 또는 강한 그라디언트로 시선 집중

**적용 포인트**: 세로형은 스크롤 방식으로 정보 전달. 각 섹션을 화면 단위로 분리.

---

### 레퍼런스 ③: 프리미엄 부동산 서비스 브랜딩 (Luxury Presence)
**출처**: Luxury Presence Brand Guide

**디자인 포인트**:
- **컬러**: Deep Navy(#0D1B2A) + Gold Accent(#C9A96E) + Pure White(#FFFFFF)
- **타이포**: 산세리프 Bold로 헤드라인 → 산세리프 Regular로 본문
- **계층**: 브랜드명(최대) → 후킹 카피(대형) → 서비스 포인트(중형) → CTA(소형)
- **신뢰 요소**: 숫자 데이터("4명 전담", "0% 수수료") 그래픽으로 강조
- **여백**: 화이트스페이스를 넉넉히 사용해 고급감 유지

**적용 포인트**: "프리미엄"과 "신뢰"를 동시에 표현하는 컬러+타이포 조합.

---

### 레퍼런스 ④: 2024-2025 광고 디자인 트렌드 — 빅타이포+하이퍼컬러
**출처**: MADTimes, Shopify, Accio

**디자인 포인트**:
- **빅타이포그래피**: 타이틀 폰트가 화면의 30~40%를 차지 → 압도적 시선 집중
- **대비**: 다크 배경 + 라이트 텍스트 대비비 최소 7:1 이상
- **강조 컬러**: 단 하나의 포인트 컬러로 CTA, 숫자 강조
- **레이아웃**: 비대칭 그리드, 텍스트가 이미지 위로 오버레이
- **트렌드**: 손글씨보다 기하학적 산세리프가 "신뢰+모던" 동시 표현

**적용 포인트**: "등기권리증" 같은 핵심 단어를 초대형으로 배치해 후킹 극대화.

---

### 레퍼런스 ⑤: 한국 모던 폰트 — Paperlogy / Pretendard
**출처**: GitHub Paperlogy, Pretendard 공식 페이지

**디자인 포인트**:
- **Paperlogy**: G Market Sans 기반, 프레젠테이션 특화, 커닝 완벽, Bold가 임팩트 있음
- **Pretendard**: 시스템 UI 대체, 가독성 최우선, 다양한 웨이트(100~900)
- **공통**: 한글+영문+숫자 모두 균형 잡힌 산세리프
- **DID 적합성**: Paperlogy ExtraBold / Pretendard Black이 원거리 가독성 최적

## 3. 종합 디자인 방향 제안

### 3-1. 컬러 팔레트 (확정 제안)

```
배경 (Primary Background):
  - 딥 네이비: #0D1B2A (메인 배경)
  - 미드나이트: #1A2E4A (그라디언트 서브)

강조 (Accent Colors):
  - 프리미엄 골드: #C9A96E (핵심 수치, CTA, 포인트)
  - 라이트 골드: #E8D5A3 (서브 강조)

텍스트 (Text Colors):
  - 순백: #FFFFFF (메인 타이틀)
  - 오프화이트: #F0EBE3 (서브 텍스트)
  - 연회색: #A8B4C2 (설명 텍스트)

포인트 (Hook Colors):
  - 에너지 오렌지: #FF6B35 (긴급성/CTA 포인트) — 선택사항
```

### 3-2. 타이포그래피 방향

| 역할 | 폰트 | 웨이트 | 크기(2160px 기준) |
|------|------|--------|-----------------|
| 메인 후킹 타이틀 | Paperlogy | ExtraBold (800) | 180~220px |
| 서브 타이틀 | Pretendard | Bold (700) | 80~100px |
| 서비스 포인트 | Pretendard | SemiBold (600) | 56~64px |
| 본문/설명 | Pretendard | Regular (400) | 40~48px |
| CTA | Pretendard | Bold (700) | 60~72px |

**메인폰트(타이틀폰트): Paperlogy ExtraBold** — 임팩트와 가독성 최적

### 3-3. 레이아웃 구성 (세로형 2160×3840px)

```
[Zone 1: 후킹 헤드라인 — 화면 상단 30%]
  - 배경: 딥 네이비 풀블리드
  - 내용: 메인 후킹 카피 초대형 타이포
  - "돈은 있는데 확신이 없어 망설이다가"
  - 포인트 컬러로 "또 버스를 놓치시겠습니까?" 강조

[Zone 2: 브랜드 정체성 — 화면 25%]
  - 내용: 바로이집 풀케어 서비스 로고/명칭
  - 서브카피: "등기 칠 때까지 고수의 1:1 전담마크"
  - 골드 언더라인으로 권위 표현

[Zone 3: 서비스 강점 — 화면 25%]
  - 4개 아이콘 + 서비스명 (세무, 법무, 협상, 인테리어)
  - 카드형 그리드 (2×2)
  - 각 카드: 아이콘 + 타이틀 + 한줄 설명

[Zone 4: CTA + 신뢰 지표 — 화면 20%]
  - 핵심 수치 강조 (숫자 크게)
  - 상담 신청 CTA 버튼 (골드 배경 + 다크 텍스트)
  - QR 코드 + 연락처
```

### 3-4. 후킹 요소 목록

1. **초대형 타이포**: 메인 카피의 핵심 단어를 화면의 40% 크기로
2. **숫자 강조**: "4명", "1:1", "100%"를 골드 컬러 + 극대형으로
3. **강한 명암 대비**: 네이비 배경에 흰 텍스트 (대비비 >10:1)
4. **골드 포인트라인**: 구역 구분 + 시선 흐름 안내
5. **그라디언트 오버레이**: 상단 濃 → 하단 淡으로 자연스러운 시선 이동
6. **카운팅 배지**: 전담팀 인원수, 성공 사례 수 등 신뢰 수치

## 4. 경쟁사 차별화 포인트

| 일반 부동산 광고 | 바로이집 풀케어서비스 광고 |
|----------------|------------------------|
| 물건 중심 (가격, 평수) | 사람 중심 (전담팀, 1:1 서비스) |
| 수동적 정보 나열 | 감정적 후킹 ("망설이다가 또 버스를...") |
| 화이트/라이트 배경 | 프리미엄 다크 배경 |
| 작은 폰트, 많은 텍스트 | 빅타이포, 핵심만 |
| 일반 광고체 | 모던 임팩트 산세리프 |

## 5. 다음 단계 (design-asset-generator) 인수인계

**생성이 필요한 에셋**:
1. 배경 그라디언트 (딥 네이비 계열)
2. 골드 라인/구분선 장식 요소
3. 서비스 아이콘 4종 (세무사, 법무사, 중개협상, 인테리어) — 라인아트 스타일
4. 신뢰 배지 (팀원수, 경력 등)
5. 집/열쇠/등기권리증 메인 비주얼 요소
6. CTA 버튼 컴포넌트

**확정 폰트**:
- 타이틀: Paperlogy ExtraBold
- 본문: Pretendard Bold / Regular
- CDN: jsDelivr Paperlogy + Google Fonts Noto Sans KR 대체 가능
