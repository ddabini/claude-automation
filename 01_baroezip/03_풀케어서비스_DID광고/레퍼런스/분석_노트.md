## 1. 공통 디자인 트렌드

### 1-1. 컬러 트렌드
수집된 레퍼런스들에서 공통적으로 발견된 색상 경향:

**다크 배경 계열 (빈도 높음)**
- 딥 네이비: #0A1628 ~ #0D1B2A — 가장 많이 등장. 권위 + 신뢰 표현
- 차콜/미드나이트: #1A1A2E ~ #1A2E4A — 세련된 다크 배경
- 순수 블랙(#000000)보다 약간 컬러가 섞인 "따뜻한 블랙" 선호

**포인트 컬러 트렌드**
- 골드/샴페인: #C9A96E ~ #D4AF37 — 럭셔리, 신뢰, 프리미엄 표현
- 오렌지-브론즈: #FF6B35 ~ #C17F3A — 에너지, 행동 유발
- 화이트/크림: #F5F0E8 ~ #FFFFFF — 대비 강화용 텍스트

**2025 트렌드 (Behance Design Trends 2025 기준)**
- 다크 블루 + 네온 오렌지 페어링 — 볼드 대비의 극단
- 보라/퍼플 계열 + 밝은 라임 — 에너지, 창의성
- 클래식 다크 + 골드 — 시간을 초월하는 프리미엄

### 1-2. 타이포그래피 트렌드
- **초대형 타이포**: 화면의 30-50% 영역을 텍스트로 채우는 "빅타이포" 트렌드 강세
- **폰트 웨이트 대비**: ExtraBold/Black 타이틀 vs Regular/Light 서브텍스트 → 드라마틱 계층 구조
- **기하학 산세리프 우세**: 세리프보다 산세리프가 "모던 럭셔리" 표현에 더 많이 쓰임
- **한국어 광고**: Paperlogy, Pretendard, Noto Sans KR 등 현대적 한글 폰트 주류

### 1-3. 레이아웃 패턴
수집된 레퍼런스에서 공통적으로 발견된 레이아웃 구조:

**세로형 포스터/배너 (우리 목적과 직접 관련)**
```
[상단 30%] 후킹 헤드라인 — 초대형 타이포, 단 1-2문장
[중단 40%] 핵심 내용 — 서비스 항목, 이점, 아이콘
[하단 30%] CTA + 신뢰 요소 — 버튼, 연락처, 수치
```

**카드/모듈형 레이아웃**
- 2×2 또는 3×2 그리드로 서비스 항목 배치
- 각 카드: 아이콘(상단) + 제목(중단) + 설명(하단)
- 카드 구분: 얇은 골드 라인 또는 미묘한 배경색 차이

**비대칭 레이아웃**
- 텍스트가 이미지 위로 오버레이되는 방식
- 왼쪽 정렬이 중앙 정렬보다 "권위 있는 느낌" 강화

### 1-4. 비주얼 장치 트렌드
- **글로우 효과**: 텍스트나 아이콘에 미묘한 발광 효과 → 디지털 화면에서 미래적 인상
- **그라디언트 오버레이**: 배경 이미지 위에 어두운 그라디언트 → 텍스트 가독성 확보
- **골드 라인/구분선**: 섹션 구분, 강조 포인트, 프리미엄 감성
- **블러 효과**: 배경 일부를 흐리게 처리해 텍스트 레이어 분리
- **카운터/배지**: 숫자 데이터 강조 ("4명 전담", "신뢰도 98%")

## 2. DID 광고 설계 원칙 (CrownTV / Screenly 가이드 기반)

### 5x5 룰 적용
- 화면당 최대 5줄
- 한 줄당 최대 5단어
- **2160px 기준 최소 폰트 크기**: 타이틀 120px 이상

### 대비 기준
- WCAG AA: 대비비 4.5:1 이상 (일반 텍스트)
- WCAG AAA: 대비비 7:1 이상 (큰 텍스트) → 원거리 가독성 목표

### 세로형(9:16) 특성
- 세로로 스크롤하듯 섹션을 분리
- 각 섹션은 "독립적으로 읽혀야" 함 (시청자가 지나칠 수 있음)
- 상단 1/3은 시선이 가장 먼저 닿는 영역 → 후킹 요소 배치 필수

## 3. 경쟁사 분석 시사점

### 일반 부동산 광고의 특징 (피해야 할 패턴)
- 물건 사진 + 가격 + 평수 위주
- 화이트/라이트 배경으로 평범한 느낌
- 텍스트 과다, 폰트 크기 작음
- 특색 없는 파란색/초록색 사용

### 바로이집이 노려야 할 차별화 방향
- **감정 유발형 후킹**: "아직 혼자 계세요?" 류의 공감 카피
- **다크 프리미엄**: 전문 서비스 = 권위 = 다크 배경
- **서비스 구체화**: 추상적 "전문가" 대신 "세무사 + 법무사 + 협상가 + 인테리어"
- **숫자로 신뢰**: "4명이 함께", "등기까지 1:1" 등 구체적 수치

## 4. 폰트 선택 근거

### 한글 메인폰트: Paperlogy ExtraBold
- G Market Sans 기반으로 프레젠테이션 특화 설계
- ExtraBold (800)는 원거리에서도 선명한 가독성
- 상업적 사용 무료, CDN 제공

### 한글 본문폰트: Pretendard Bold/Regular
- 시스템 UI 대체 폰트로 모든 환경에서 균등한 렌더링
- 웨이트 100~900 전 범위 지원 → 계층 구조 자유롭게 설계
- 영문과의 조화 우수

### 영문 혼용 시: 동일 폰트 유지 or Inter
- Paperlogy/Pretendard의 영문 자형이 충분히 깔끔함
- 필요 시 Inter Bold와 페어링 가능
