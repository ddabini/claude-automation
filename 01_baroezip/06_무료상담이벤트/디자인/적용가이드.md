
## design-figma 전달 메모

- **다음 단계**: HTML 조립 (`design-figma` 에이전트)
- **포스터 크기**: 1080×1080px (인스타그램 피드 정방형)
- **배경**: #0D1B2A 단색 배경 (이미지 없음 — 순수 타이포그래피 포스터)
- **폰트 로드**: Paperlogy woff2 CDN + Pretendard CSS CDN 두 개 모두 필수
- **레이아웃**: 3존 구조 (로고20% / 헤드라인40% / 서브·바디·CTA40%)

- **코너 장식**: tl.svg (좌상단 40, 40) + br.svg (우하단 40, 40) 절대 위치
- **구분선**: Zone 경계 y=180, y=540 두 곳에 각 1개씩 배치 (x=140, width=800)
