# 타깃 페르소나 설계

> 바로이집의 핵심 고객 유형을 구체적으로 정의
> 작성일: 2026-02-26 | 작성: strategy-planning 에이전트

---

## 1. 왜 페르소나가 필요한가

"2030~40대 실수요자"라는 타깃은 너무 넓습니다. 같은 30대 부부라도 첫 전세를 구하는 신혼부부와, 아이 학군 때문에 이사하는 부부는 완전히 다른 사람입니다. 각각의 고객이 **무엇 때문에 밤에 잠을 못 자는지**, **어떤 말에 귀를 기울이는지** 알아야 정확한 서비스를 만들 수 있습니다.

## 2. 페르소나 1: 김서연 (32세) — "첫 전세 신혼부부"

### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 김서연 (가명) |
| 나이 | 32세 |
| 직업 | IT 기업 마케터 (연봉 4,500만원) |
| 가족 | 남편 (34세, 개발자, 연봉 5,500만원), 결혼 1년차 |
| 거주 | 서울 마포구 원룸 (월세 65만원) |
| 목표 | 신혼 전세 구하기 (예산 3~4억, 서울 또는 경기 남부) |

### Pain Point (고통 포인트)

1. **전세사기 공포**: 뉴스에서 전세사기 피해 사례를 자주 접함. 당근 부동산 직거래는 무료지만 "혹시 사기면 어쩌지?" 불안[27]
2. **시간 부족**: 맞벌이라 평일에 집을 보러 다닐 수 없음. 주말에 몰아서 봐야 하는데, 좋은 매물은 금방 사라짐
3. **정보 비대칭**: 전세자금 대출 조건, 보증보험 가입 여부, 등기부 해석 방법을 모름. 중개사 말을 어디까지 믿어야 할지 모름
4. **예산 불확실**: 전세자금 대출이 얼마까지 나오는지, 보증금 + 이사비 + 가전 구입까지 합치면 총 얼마가 필요한지 감이 안 잡힘

### Need (필요)

| 우선순위 | 필요 | 구체적 기대 |
|---------|------|-----------|
| 1순위 | **안전 확인** | "이 집이 안전한지 한눈에 보여줘" |
| 2순위 | **시간 절약** | "내 조건에 맞는 매물만 추려서 보여줘" |
| 3순위 | **전문가 조언** | "처음이라 뭘 확인해야 하는지 알려줘" |
| 4순위 | **비용 예측** | "대출부터 이사비까지 총 비용을 알려줘" |

### 행동 패턴

- **정보 수집**: 네이버 카페 "부동산 스터디", 유튜브 "부읽남", "집코노미" 시청
- **매물 검색**: 직방/다방 앱 동시 사용, 호갱노노로 실거래가 확인
- **의사결정**: 남편과 함께 결정, 부모님 의견도 참고
- **불안 해소**: 블로그에서 "전세사기 체크리스트" 검색, 커뮤니티에 "이 매물 괜찮을까요?" 질문
- **이용 시간대**: 출퇴근 지하철(08:00~09:00), 점심시간(12:00~13:00), 퇴근 후(21:00~23:00)

### 바로이집이 이 고객에게 해줄 수 있는 것

```
김서연의 불안: "전세사기 당하면 어쩌지..."
바로이집의 답: AI가 등기부/시세/보증보험 가입 가능 여부를 자동 분석해서
             위험도를 점수로 보여드립니다. 90점 이상 매물만 추천합니다.

김서연의 불안: "평일에 시간이 없어..."
바로이집의 답: 전담 중개사가 평일에 미리 매물을 확인하고,
             주말에 효율적으로 볼 수 있도록 동선과 스케줄을 잡아드립니다.

김서연의 불안: "처음이라 뭘 봐야 하는지..."
바로이집의 답: 방문 동행하며 체크리스트를 함께 확인하고,
             계약서 검토부터 이사 당일까지 옆에서 안내합니다.
```

### 전환 트리거 (이 고객이 바로이집을 선택하는 순간)

- 친구가 "나 바로이집으로 전세 구했는데, 전담 중개사가 다 알려줘서 편했어" 후기
- 인스타그램에서 "전세사기 위험도 무료 진단" 광고를 봤을 때
- 네이버 카페에서 "바로이집 이용 후기" 글을 읽었을 때

## 3. 페르소나 2: 박준혁 (38세) — "학군 이사 워킹맘/대디"

### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 박준혁 (가명) |
| 나이 | 38세 |
| 직업 | 제조업 과장 (연봉 6,000만원) |
| 가족 | 아내 (36세, 간호사, 연봉 4,500만원), 아들 (6세, 내년 초등 입학) |
| 거주 | 경기 성남시 분당구 아파트 (전세 4.5억) |
| 목표 | 아들 초등 입학 전 학군 좋은 지역으로 매매/전세 이동 (예산 6~8억) |

### Pain Point (고통 포인트)

1. **학군 정보 부족**: 어떤 학교가 좋은지, 학원가 접근성은 어떤지 객관적 데이터가 없음. 맘카페 정보는 주관적
2. **매매 vs 전세 판단 어려움**: 지금 사야 하는지, 전세로 더 기다려야 하는지. AI 시세 예측 정확도가 94.7%라지만[25] 해석이 어려움
3. **복잡한 절차**: 기존 전세 만기 처리 + 새 집 계약 + 이사 + 아이 전학/유치원 변경... 동시에 처리할 일이 너무 많음
4. **부부 의견 불일치**: 아내는 "학군 최우선", 본인은 "통근 거리 중요". 타협점을 찾기 어려움

### Need (필요)

| 우선순위 | 필요 | 구체적 기대 |
|---------|------|-----------|
| 1순위 | **학군 + 생활 인프라 데이터** | "이 동네 학교 평가, 학원가, 소아과 다 보여줘" |
| 2순위 | **매매/전세 의사결정 지원** | "지금 시세 추세와 적정가를 분석해줘" |
| 3순위 | **전 과정 관리** | "기존 전세 만기 처리부터 새 집 입주까지 스케줄 잡아줘" |
| 4순위 | **부부 합의 도구** | "내 조건과 아내 조건을 동시에 만족하는 매물 추천" |

### 행동 패턴

- **정보 수집**: 맘카페(맘스홀릭, 지역 맘카페), 호갱노노 실거래가 확인, 학군 관련 블로그
- **매물 검색**: 네이버 부동산 + 직방 병행, 주말에 부부 함께 매물 방문
- **의사결정**: 아내와 함께 결정, 엑셀로 매물 비교표 작성, 최종 결정까지 2~3개월
- **불안 해소**: "OO 아파트 단지 어떤가요?" 맘카페 질문, 주변 지인 경험담 수집
- **이용 시간대**: 아이 재운 후 밤 시간(22:00~24:00), 주말 오전(10:00~12:00)

### 바로이집이 이 고객에게 해줄 수 있는 것

```
박준혁의 고민: "학군 좋으면서 통근도 편한 곳이 있을까..."
바로이집의 답: 학군 등급 + 남편 통근 시간 + 아내 통근 시간을 동시에
             필터링해서 "부부 맞춤 추천 매물"을 보여드립니다.

박준혁의 고민: "기존 전세 만기 + 새 집 계약 타이밍이..."
바로이집의 답: 전담 중개사가 기존 전세 만기 처리 → 새 집 계약 → 이사 →
             전학/전입 신고까지 전 과정의 타임라인을 관리합니다.

박준혁의 고민: "지금 매매가 맞는 건지..."
바로이집의 답: AI 시세 분석 기반으로 해당 단지의 가격 추세, 적정가 범위,
             전세가율을 정리해서 "매매 vs 전세" 판단 자료를 제공합니다.
```

### 전환 트리거

- 맘카페에서 "바로이집으로 학군 이사했는데, 학교 정보까지 정리해줬어" 후기
- "학군 맞춤 매물 추천" 무료 상담 이벤트
- 지인 소개 ("우리도 바로이집 썼는데 진짜 편했어")

## 4. 페르소나 3: 이수진 (42세) — "갈아타기 매매"

### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 이수진 (가명) |
| 나이 | 42세 |
| 직업 | 금융사 차장 (연봉 7,500만원) |
| 가족 | 남편 (44세, 자영업), 딸 (12세), 아들 (9세) |
| 거주 | 서울 노원구 중계동 아파트 (매매 6.5억, 본인 소유) |
| 목표 | 강남/서초/송파 갈아타기 매매 (예산 12~15억) |

### Pain Point (고통 포인트)

1. **타이밍 리스크**: 기존 아파트를 먼저 팔면 갈 곳이 없고, 먼저 사면 자금이 부족. 매도-매수 타이밍 맞추기가 최대 난제
2. **가격 협상 불안**: 수억원 단위 거래에서 적정가를 판단하기 어려움. "더 깎을 수 있었던 건 아닌지" 항상 남는 아쉬움
3. **세금 복잡성**: 양도세, 취득세, 중개수수료, 이사비 등 총비용 산정이 복잡
4. **중개사 신뢰 문제**: 이전 거래에서 중개사가 매수인 편을 든 경험. 중개사가 내 편인지 확신이 없음

### Need (필요)

| 우선순위 | 필요 | 구체적 기대 |
|---------|------|-----------|
| 1순위 | **매도-매수 동시 관리** | "팔기와 사기를 동시에 전략적으로 관리해줘" |
| 2순위 | **가격 협상 전문가** | "적정가 근거를 데이터로 보여주고, 협상을 대행해줘" |
| 3순위 | **세금/비용 시뮬레이션** | "양도세, 취득세, 총비용을 한번에 계산해줘" |
| 4순위 | **전적으로 내 편** | "매도 중개사와 매수 중개사를 분리하지 않아도 신뢰할 수 있는 중개" |

### 행동 패턴

- **정보 수집**: 네이버 부동산 시세 확인, 국세청 양도세 계산기, 부동산 유튜브
- **매물 검색**: 네이버 부동산 중심, 지인 네트워크 활용
- **의사결정**: 남편과 상의 후 본인이 최종 결정, 결정까지 3~6개월
- **신뢰 기준**: 데이터 기반 근거를 제시하는 전문가를 신뢰, 감성적 설득에는 반감
- **이용 시간대**: 점심시간(12:00~13:00), 퇴근 후(19:00~21:00)

### 바로이집이 이 고객에게 해줄 수 있는 것

```
이수진의 고민: "팔고 사는 타이밍을 어떻게..."
바로이집의 답: 전담 중개사가 매도-매수 전 과정을 하나의 프로젝트로 관리합니다.
             기존 집 매도 전략(가격/시기) + 신규 집 매수 전략을 동시에 설계합니다.

이수진의 고민: "이 가격이 적정한 건지..."
바로이집의 답: AI 시세 분석 + 최근 3개월 동일 단지/평형 실거래 데이터를 기반으로
             "적정 매매가 범위"를 제시하고, 협상 시 근거 자료로 활용합니다.

이수진의 고민: "세금까지 합치면 총 얼마나 드는 거지..."
바로이집의 답: 양도세 + 취득세 + 중개수수료 + 이사비 + 인테리어비까지
             총비용 시뮬레이션을 제공합니다.
```

### 전환 트리거

- "갈아타기 전략 무료 상담" 서비스
- 지인이 "바로이집 전담 중개사가 매도-매수 동시에 관리해줘서 편했다" 추천
- "AI 적정가 분석 + 세금 시뮬레이션" 무료 리포트 제공

## 6. 공통 인사이트

3개 페르소나를 관통하는 핵심 인사이트:

1. **불안이 가장 큰 동기**: 전세사기, 가격 손해, 절차 실수에 대한 불안이 모든 페르소나의 출발점
2. **시간이 없다**: 맞벌이 부부는 주말/퇴근 후에만 매물을 볼 수 있음. "내 시간을 아껴주는 서비스"가 핵심
3. **데이터를 원한다**: 감에 의존하지 않고 AI/데이터 기반 근거를 원함. AI 시세 예측이 94.7% 정확도를 달성한 시장[25]에서 데이터 없는 서비스는 신뢰받기 어려움
4. **사람이 필요하다**: 앱만으로는 부족함. 전문가의 동행, 조언, 관리를 원함. 기술과 사람의 조합이 바로이집의 포지션
