## 2. 공통 트렌드 분석

### 2-1. 색상 트렌드

| 트렌드 | 설명 |
|--------|------|
| **라이트 테마 우세** | 부동산/교육/핀테크 업종은 화이트 또는 밝은 배경 기반이 대다수 |
| **2-3색 제한** | 한 카드 내 색상은 2-3개로 제한하여 깔끔한 인상 유지 |
| **포인트 컬러 강조** | 배경은 밝게, CTA나 핵심 문구에만 강렬한 포인트 컬러 적용 |
| **톤온톤/톤인톤** | 같은 계열 색상의 명도 차이로 세련된 느낌 연출 |
| **그라디언트 활용** | 단색보다 미세한 그라디언트(특히 블루-퍼플, 블루-그린 계열)가 트렌드 |

### 2-2. 레이아웃 트렌드

| 트렌드 | 설명 |
|--------|------|
| **4:5 비율 (1080x1350px)** | 인스타그램 피드에서 면적을 최대로 차지하는 세로형 포맷이 표준 |
| **센터 정렬** | 핵심 메시지와 CTA를 화면 중앙에 배치하는 구조가 주류 |
| **충분한 여백** | 글자와 요소 사이에 넉넉한 여백으로 가독성 확보 |
| **3장 구성의 정석** | 1장(훅/표지) + 2장(혜택/참여방법) + 3장(CTA/팔로우 유도) |
| **시밀리스 캐러셀** | 슬라이드 간 이미지가 연결되어 스와이프를 유도하는 디자인 인기 |

### 2-3. 타이포그래피 트렌드

| 트렌드 | 설명 |
|--------|------|
| **오버사이즈 타이포** | 첫 장의 훅 문구는 크고 굵은 폰트로 시선 강탈 |
| **2단계 폰트 위계** | 제목(Bold/ExtraBold) + 본문(Regular/Medium)의 명확한 구분 |
| **한글 폰트** | Pretendard, Noto Sans KR, Spoqa Han Sans 등 가독성 높은 산세리프 폰트 |
| **강조 포인트** | 핵심 숫자나 키워드만 컬러 변경 또는 크기 확대로 강조 |
| **짧은 문장** | 한 줄에 7-10자 이내의 짧고 임팩트 있는 문장 |

### 2-4. CTA(Call-To-Action) 배치 패턴

| 패턴 | 설명 |
|------|------|
| **마지막 장 전용 CTA** | 3장째에 "지금 팔로우하세요", "팔로우하고 혜택 받기" 등 명확한 행동 유도 |
| **화살표/손가락 아이콘** | 스와이프 유도를 위한 방향 아이콘을 슬라이드 우측에 배치 |
| **참여 조건 넘버링** | STEP 1, STEP 2, STEP 3 등 참여 방법을 숫자로 나열 |
| **프로필 링크 유도** | "프로필 링크 클릭" 또는 "@계정명 팔로우" 형태의 직접적 행동 지시 |
| **기한 표시** | "~월 ~일까지", "선착순 OO명" 등 긴급성(FOMO) 유발 |

## 3. 바로이집 추천 색상 팔레트

바로이집의 브랜드 특성(부동산 AI 에이전트 + 투자 강의, 전문적이면서 친근한 톤, 라이트 테마)을 고려한 추천 팔레트입니다.

### Option A: 클린 블루 (신뢰 + 전문성)

```
배경(Primary BG):    #FFFFFF (화이트)
보조 배경(Sub BG):    #F5F7FA (밝은 그레이블루)
메인 컬러(Primary):   #2563EB (로열 블루)
서브 컬러(Secondary): #60A5FA (라이트 블루)
강조 컬러(Accent):    #F59E0B (앰버/골드)
텍스트(Text):         #1E293B (다크 네이비)
서브 텍스트(Sub):     #64748B (미디엄 그레이)
```

- **이유**: 블루 계열은 부동산/금융/테크 업종에서 "신뢰"와 "전문성"을 전달. 골드 포인트로 "프리미엄"과 "혜택" 강조.

### Option B: 딥 네이비 + 민트 (모던 + 혁신)

```
배경(Primary BG):    #FFFFFF (화이트)
보조 배경(Sub BG):    #F0F9FF (아이스 블루)
메인 컬러(Primary):   #0F172A (딥 네이비)
서브 컬러(Secondary): #06B6D4 (시안/민트)
강조 컬러(Accent):    #8B5CF6 (바이올렛)
텍스트(Text):         #0F172A (딥 네이비)
서브 텍스트(Sub):     #475569 (슬레이트 그레이)
```

- **이유**: AI/테크 느낌을 살리면서도 민트 계열로 친근함 부여. 바이올렛 포인트로 혁신적 이미지.

### Option C: 그린 + 화이트 (성장 + 투자)

```
배경(Primary BG):    #FFFFFF (화이트)
보조 배경(Sub BG):    #F0FDF4 (아이스 그린)
메인 컬러(Primary):   #16A34A (에메랄드 그린)
서브 컬러(Secondary): #86EFAC (라이트 그린)
강조 컬러(Accent):    #EA580C (오렌지)
텍스트(Text):         #1A2E1A (다크 그린)
서브 텍스트(Sub):     #6B7280 (쿨 그레이)
```

- **이유**: 그린은 "성장", "투자 수익", "부동산 가치 상승"을 연상. 오렌지 포인트로 에너지와 행동 유도.

### 최종 추천: **Option A (클린 블루)** 또는 **Option B (딥 네이비 + 민트)**

- 바로이집이 "부동산 AI 에이전트"라는 테크 기반 브랜드이므로, 블루 계열이 신뢰와 기술력을 동시에 전달
- 라이트 테마 선호에도 가장 잘 맞음

## 4. 추천 레이아웃 패턴 (카드뉴스 3장 구성)

### 1장: 훅(Hook) - 시선 잡기

```
┌─────────────────────┐
│                     │
│    [포인트 아이콘]    │
│                     │
│  ■■■■■■■■■■■■■■■■  │ ← 메인 카피 (오버사이즈, 볼드)
│  ■■■■■■■■■■■■■■■■  │   예: "팔로우만 하면"
│                     │
│   서브 카피 1줄      │ ← 보조 설명 (작은 폰트)
│                     │
│  ──────────────────  │
│  @baroezip 로고/마크  │ ← 브랜드 식별 요소
│                     │
│         >>>         │ ← 스와이프 유도 아이콘
└─────────────────────┘
```

**핵심 원칙**: 3초 안에 관심을 끌 수 있는 짧고 강렬한 문구 + 브랜드 아이덴티티

### 2장: 혜택/참여방법

```
┌─────────────────────┐
│                     │
│  [이벤트 혜택 제목]   │ ← Bold, 포인트 컬러
│                     │
│  ┌───────────────┐  │
│  │ STEP 1        │  │ ← 팔로우
│  │ 📌 @baroezip  │  │
│  │   팔로우하기    │  │
│  ├───────────────┤  │
│  │ STEP 2        │  │ ← 좋아요 또는 댓글
│  │ ❤️ 이 게시물에 │  │
│  │   좋아요 누르기 │  │
│  ├───────────────┤  │
│  │ STEP 3        │  │ ← 친구 태그
│  │ 👥 친구 1명    │  │
│  │   태그하기     │  │
│  └───────────────┘  │
│                     │
│         >>>         │
└─────────────────────┘
```

**핵심 원칙**: 참여 방법을 3단계 이내로 단순화. 각 단계를 시각적으로 분리.

### 3장: CTA + 마무리

```
┌─────────────────────┐
│                     │
│   🎁 당첨 혜택      │ ← 경품/혜택 강조
│                     │
│  ┌───────────────┐  │
│  │  경품 이미지   │  │ ← 구체적 혜택 시각화
│  │  또는 설명     │  │
│  └───────────────┘  │
│                     │
│  📅 ~월 ~일까지     │ ← 기한 + 긴급성
│  📌 선착순 OO명     │
│                     │
│  ━━━━━━━━━━━━━━━━━  │
│  [ 지금 팔로우하기 ] │ ← CTA 버튼 스타일
│                     │
│  @baroezip          │ ← 계정명 재노출
│  로고               │
└─────────────────────┘
```

**핵심 원칙**: 혜택을 구체적으로 제시 + 기한으로 긴급성 유발 + 명확한 CTA

## 6. 바로이집 카드뉴스 디자인 제작 시 핵심 체크리스트

- [ ] 규격: 1080x1350px (4:5 비율) 또는 1080x1080px (1:1)
- [ ] 색상: 메인 2색 + 포인트 1색 이내로 제한
- [ ] 테마: 라이트(화이트 기반) 배경
- [ ] 폰트: 제목-본문 2단계 위계, 산세리프 기반
- [ ] 1장(훅): 3초 내 관심 끄는 짧고 강렬한 문구
- [ ] 2장(참여방법): STEP 1-2-3 형태로 단순 명료하게
- [ ] 3장(CTA): 구체적 혜택 + 기한 + 명확한 행동 유도
- [ ] 브랜드: @baroezip 계정명 + 로고 매 장 노출
- [ ] 스와이프: 슬라이드 간 연결성(시각적 힌트) 포함
- [ ] 모바일 최적화: 작은 화면에서도 텍스트 읽힘 확인
