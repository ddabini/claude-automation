# 바로이집 인스타그램 팔로우 유도 캠페인 카드뉴스 기획안

**작성일**: 2026-02-24
**목적**: 신규 팔로워 확보
**타깃**: 부동산 투자에 관심 있는 2030~4050 직장인/투자자

---

## 1. 이벤트 컨셉

**한 줄 컨셉**
> "지금 부동산 살 때 맞아요? — 팔로우만 해도 AI가 먼저 알려드립니다"

**이벤트 메커니즘**
- 참여 방법: 팔로우 + 게시물 저장 (2단계, 3초 완료)
- 당첨 혜택: 추첨 3명 — 바로이집 부동산 투자 강의 무료 수강권 증정
- 진입 장벽 최소화로 참여율 극대화, 강의 수강권으로 고관여 타깃 유입

---

## 2. 카드별 구성

### 1장 — 표지 / 훅

| 항목 | 내용 |
|------|------|
| 목적 | 피드 스크롤을 멈추게 하는 강한 첫 화면 |
| 핵심 감정 | 공감(내 고민을 알아준다) + 호기심(AI가 뭘 해준다는 거지?) |
| 헤드카피 | 지금 부동산 살 때 맞아요? |
| 서브카피 | AI가 매일 답해드립니다 |
| 보조 문구 | 팔로우하면 제일 먼저 알려드려요 |
| 브랜드 | 바로이집 |

### 2장 — 본문 / 혜택 설명

| 항목 | 내용 |
|------|------|
| 목적 | "왜 팔로우해야 하는가"에 혜택으로 명확히 답하기 |
| 핵심 감정 | 기대(이걸 알면 나도 앞서갈 수 있다) |
| 소제목 | 바로이집을 팔로우해야 하는 이유 3가지 |
| 혜택 1 | AI가 분석한 부동산 투자 인사이트 — 매주 무료 제공 |
| 혜택 2 | 입지·시세·갭투자 핵심 정보 — 복잡한 것을 한 장으로 |
| 혜택 3 | 팔로워 한정 강의 할인 & 무료 수강 이벤트 정기 진행 |
| 하단 문구 | 혼자 공부하지 마세요. 바로이집이 옆에 있습니다. |

### 3장 — CTA / 참여 방법

| 항목 | 내용 |
|------|------|
| 목적 | 지금 바로 팔로우 버튼을 누르게 만들기 |
| 핵심 감정 | 긴박감(지금 참여해야 혜택을 받는다) |
| 이벤트 제목 | 팔로우 이벤트 — 강의 무료 수강권 드립니다 |
| 참여 방법 | ① 바로이집 팔로우 ② 이 게시물 저장 |
| 당첨 혜택 | 추첨 3명, 바로이집 부동산 투자 강의 무료 수강권 |
| 마무리 카피 | 지금 팔로우하고 저장하세요. 부동산은 타이밍입니다. |

## 3. 카드별 텍스트 카피

### 1장 — 표지/훅

```
지금 부동산 살 때 맞아요?
AI가 매일 답해드립니다.

팔로우하면 제일 먼저 알려드려요.

바로이집
```

**대안 헤드카피 (A/B 테스트용)**
- "아직도 발품 팔아요? AI는 3초면 됩니다."
- "매달 수천 명이 바로이집 AI로 부동산을 분석하는 이유"

---

### 2장 — 본문/혜택

```
바로이집을 팔로우해야 하는 이유 3가지

01  AI 부동산 인사이트 — 매주 무료 제공
    시장 흐름, 입지 분석, 갭투자 타이밍을 AI가 정리해 드립니다.

02  복잡한 부동산 정보 — 한 장으로 정리
    공시가·실거래가·전세가율, 헷갈리지 않게 핵심만.

03  팔로워 한정 강의 혜택
    무료 수강 이벤트, 할인 정보 — 팔로워에게만 드립니다.

혼자 공부하지 마세요.
바로이집이 옆에 있습니다.
```

---

### 3장 — CTA/참여 방법

```
팔로우 이벤트

강의 무료 수강권을 드립니다.

참여 방법
① 바로이집 팔로우
② 이 게시물 저장

추첨 3명
바로이집 부동산 투자 강의 무료 수강권 증정

마감: [날짜] 자정까지   ← 확인 필요: 실제 이벤트 마감일
당첨 발표: [발표일] 스토리 공지   ← 확인 필요: 발표일

지금 팔로우하고 저장하세요.
부동산은 타이밍입니다.

@baroezip
```

## 4. 디자인 방향 키워드

| 항목 | 방향 |
|------|------|
| 색상 톤 | 딥 네이비(#0D1B2A) + 골드/앰버(#F4A300) — 신뢰감 + 프리미엄 |
| 배경 처리 | 다크 배경 + 밝은 텍스트 — 피드 내 시선 차별화 |
| 타이포 | Pretendard Bold 계열 — 굵고 빠르게 읽히는 구조 |
| 레이아웃 | 중앙 정렬, 여백 충분 확보 — 고급스럽고 밀리지 않는 느낌 |
| 그래픽 요소 | AI 데이터 시각화 그래픽 또는 건물 실루엣 (사람 사진보다 그래픽 선호) |
| 분위기 키워드 | 신뢰 / 데이터 / 프리미엄 / 명확 |
