  • AI 영상 생성 API 시장은 2025~2026년 급성장 — 10초짜리 영상 생성 비용이 $0.25~$5 수준으로 실용화 단계 진입. 가성비 1위는 Kling($0.029/초), 품질 1위는 Runway Gen-4.5(Elo 1,247점)
  • Google Veo 3.1이 유일하게 네이티브 4K + 오디오 동시 생성을 지원하나 비용($0.35~0.50/초)이 높아 프로덕션 환경에서만 권장. Sora 2는 API 접근이 여전히 제한적

  • 브라우저 영상 처리 기술 분기점 도달 — FFmpeg.wasm(CPU 전용, 25fps)보다 WebCodecs가 8배 빠른 200fps. WebGPU는 2025년 11월 4대 브라우저 공식 지원으로 AI 추론 19~100배 가속
  • CapCut(소셜 특화)·Vrew(자막 특화)·Descript(트랜스크립트 특화)·InVideo(AI 생성 특화)로 경쟁사 포지션이 명확히 분리 — 통합 AI 생성+편집 경험이 블루오션

3-2. FFmpeg.wasm

GitHub 34,000+ Stars를 보유한 가장 널리 사용되는 브라우저 영상 처리 라이브러리입니다. 트랜스코딩, 자르기, 필터 적용 등 FFmpeg의 모든 기능을 브라우저에서 실행합니다.

npm 설치 SharedArrayBuffer 필요 COOP/COEP 헤더 설정 필요 멀티스레드 지원 트랜스코딩

FFmpeg.wasm vs WebCodecs 실제 성능 비교 (1080p, MacBook Pro M2 기준)

FFmpeg.wasm 인코딩 속도25 FPS (CPU 100%)
WebCodecs 인코딩 속도200 FPS (CPU 15%)
FFmpeg.wasm 메모리2GB+
WebCodecs 메모리훨씬 적음

Electron 데스크톱 앱 확장 시 고려사항

5-1. GPU 가속 활용 방법

Electron 앱은 Chromium 기반이므로 웹 기술(WebGL, WebGPU)을 그대로 활용할 수 있습니다. 추가로 Node.js 네이티브 모듈을 통해 더 강력한 GPU 접근이 가능합니다.

접근 방법GPU 접근 수준구현 복잡도권장 사용 사례
WebGPU (Renderer Process) 중상 (Metal/DX12/Vulkan 추상화) 중간 AI 추론, 실시간 필터
WebGL (Renderer Process) 중간 중간 영상 이펙트, 색보정
ONNX Runtime Node (Main Process) 높음 (CUDA/CoreML 직접) 높음 로컬 AI 모델 실행
FFmpeg 네이티브 바이너리 높음 (H/W 인코더) 중간 고속 트랜스코딩 (nvenc, videotoolbox)
Python 자식 프로세스 완전 (모든 GPU 라이브러리) 높음 로컬 AI 모델, 복잡한 처리
💡 핵심 차이점 웹 앱에서는 FFmpeg.wasm(CPU 전용)을 사용해야 하지만, Electron에서는 FFmpeg 네이티브 바이너리를 child_process로 실행할 수 있습니다. macOS의 VideoToolbox, NVIDIA의 NVENC 등 하드웨어 가속 인코더를 활용해 10배 이상 빠른 인코딩이 가능합니다.

5-2. 로컬 AI 모델 실행 가능성

ONNX Runtime Web/Node 실용적
  • 백엔드 (Electron Main)onnxruntime-node
  • 백엔드 (Renderer)onnxruntime-web + WebGPU
  • GPU EP (Node.js)CUDA, DirectML, CoreML
  • 지원 모델Whisper, SAM, SDXL-Turbo 등
  • SAM 가속인코더 19x (WebGPU)
로컬 AI 활용 사례 권장
  • 자막 생성Whisper.cpp (완전 로컬)
  • 배경 제거REMBG / SAM (ONNX)
  • 업스케일링Real-ESRGAN (ONNX)
  • 씬 감지PySceneDetect / FFmpeg
  • 영상 분류CLIP (ONNX)
✅ 제언 — 하이브리드 아키텍처 비용이 적게 드는 AI 기능 (자막, 배경 제거, 업스케일)은 로컬 ONNX 모델로 처리하고, 고품질 AI 영상 생성은 클라우드 API(Runway/Kling)를 호출하는 하이브리드 아키텍처를 권장합니다. 사용자가 오프라인이거나 비용을 절약하고 싶을 때 로컬 모드로 자동 전환하는 UX가 차별점이 될 수 있습니다.
