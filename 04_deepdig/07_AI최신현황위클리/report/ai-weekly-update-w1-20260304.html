Claude (Anthropic)

🟣
Claude 3.7 Sonnet
Anthropic · 2025년 2월 24일 출시
현재 버전: claude-3-7-sonnet-20250219 (API)

Claude 3.x 계열의 마지막 대형 모델. 하이브리드 추론(Hybrid Reasoning) 기능을 처음 도입하여 '확장 사고(Extended Thinking) 모드'로 복잡한 문제를 단계별로 추론합니다. [1]

  • 확장 사고 모드: 응답 전에 내부적으로 추론 과정을 거쳐 정확도 향상
  • 코딩 능력 대폭 강화 — 당시 벤치마크 1위 수준
  • 불필요한 거절 45% 감소 (안전성과 유용성 균형 개선)
  • 컨텍스트 창: 128K 토큰 (표준), 1M 토큰 (베타)
API 가격: 입력 $3 / 출력 $15 (100만 토큰 기준, 확장 사고 포함 동일 [1])
🔵
Claude Opus 4.6
Anthropic · 2026년 2월 5일 출시
현재 버전: claude-opus-4-6 (최신 Opus)

Anthropic 최고 지능 모델의 최신판. 에이전트 팀 지원과 Claude in PowerPoint(연구 미리보기) 기능이 핵심 추가 사항입니다. [2][3]

  • 1M 토큰 컨텍스트 창 (베타) — 초대형 코드베이스·문서 처리 가능
  • 적응형 사고(Adaptive Thinking): 상황에 맞게 추론 깊이를 자동 조절
  • '노력(Effort) 제어' 옵션: 개발자가 지능·속도·비용 트레이드오프를 직접 설정
  • Claude in Excel 대폭 개선 + Claude in PowerPoint 연구 미리보기
  • 대형 코드베이스 탐색, 코드 리뷰·디버깅 능력 강화
"Performance that would have previously required reaching for an Opus-class model — including on real-world, economically valuable office tasks — is now available with Sonnet 4.6." — Anthropic 공식 발표, 2026-02-17 [4]
API 가격(표준): 입력 $5 / 출력 $25 (100만 토큰 기준) [5]
고속 모드(Turbo): 입력 $30 / 출력 $150 — 표준 대비 6배 높은 가격, 응답 속도 최대 2.5배 향상 [6]
🟢
Claude Sonnet 4.6
Anthropic · 2026년 2월 17일 출시 — 현재 기본 모델
현재 버전: claude-sonnet-4-6 (대부분 사용자 기본값)

Anthropic이 "성능·속도·비용 최적의 균형"으로 내세우는 주력 모델. 처음으로 Sonnet 계열이 이전 세대 Opus를 코딩 평가에서 앞선 이정표입니다. [4][7]

  • 코딩, 컴퓨터 조작(Computer Use), 디자인, 지식 작업 전반 향상
  • 이전 세대 Opus 4.5보다 코딩 평가에서 우수한 성능 — 중간 모델로 최고 성능 달성
  • 1M 토큰 컨텍스트 창 (베타 헤더 사용 시)
  • Sonnet 4.5와 동일 가격으로 더 높은 성능 제공 [7]
API 가격: Sonnet 4.5와 동일 (Opus보다 낮은 티어, 정확 단가: $3 입력 / $15 출력 100만 토큰 기준) [4]
인사이트

Anthropic은 2026년 2월 한 달에만 Opus 4.6(2/5)과 Sonnet 4.6(2/17)을 연달아 출시했습니다. 출시 속도가 경쟁사 대비 빠른 편입니다. 특히 Sonnet이 Opus 수준 성능에 도달한 것은 "AI의 상향 평준화"를 의미합니다. 중간 가격으로 최상 성능을 낼 수 있다는 것은 기업 도입 비용 절감에 직접적으로 연결됩니다.

Claude Code

⚡
Claude Code 2.1
Anthropic · CLI 코딩 에이전트 도구
현재 버전: 2.1.66 (최신, 2026-03 기준)

터미널에서 직접 실행하는 Anthropic의 코딩 에이전트 CLI 도구. 2026년 초 2.1 버전 출시로 단일 에이전트에서 멀티 에이전트 오케스트레이션 플랫폼으로 진화했습니다. 1,096개 커밋이 포함된 대규모 업데이트입니다. [8][9]

2.1.0 핵심 신기능 (2026년 1월)

  • 서브에이전트(Subagents): 복잡한 작업을 여러 하위 에이전트에 분산 처리
  • 스킬 핫 리로드(Skills Hot Reload): 실행 중인 에이전트에 새 스킬을 즉시 추가
  • 커스텀 에이전트 지원 (/ 명령어로 호출): 사용자 정의 에이전트를 슬래시 명령으로 직접 실행
  • 자동 메모리(Auto-memory): Claude가 유용한 컨텍스트를 자동 저장, /memory로 관리
  • Shift+Enter 줄바꿈: 별도 설정 없이 즉시 사용 가능
  • /copy 명령: 코드 블록 또는 전체 응답을 대화형으로 선택해서 복사
  • 에이전트 frontmatter에 hooks 직접 추가 가능
  • 도구 권한 와일드카드 지원: 예) Bash(*-h*)

2.1 추가 업데이트 (2026년 2월)

  • 에이전트 워크트리 격리: 에이전트가 독립된 git worktree에서 안전하게 실행
  • WorktreeCreate / WorktreeRemove 훅 지원
  • claude agents CLI 명령어 추가
  • Claude Opus 4.6 1M 토큰 컨텍스트 지원
  • 새로운 claude auth login/status/logout CLI 서브커맨드
  • Windows ARM64 네이티브 바이너리 지원
  • /teleport: 세션을 http://claude.ai/code로 전송
  • HTTP 훅: JSON을 URL로 POST 방식 전달
  • Linux/Mac에서 BashTool 샌드박스 모드
인사이트

Claude Code는 2025년 2월 Claude 3.7 Sonnet 출시와 함께 처음 공개된 이래 약 1년 만에 "코딩 도우미"에서 "멀티 에이전트 오케스트레이터"로 위상이 완전히 바뀌었습니다. 특히 에이전트 워크트리 격리와 서브에이전트 기능은 엔터프라이즈 개발팀이 AI를 병렬로 활용하는 방식을 열어줍니다. Cursor, Windsurf와의 직접 경쟁 구도가 형성되고 있습니다.

OpenAI

GPT-5 (2025.8.7): 코딩, 수학, 시각 인식, 건강 등 전 분야 최고 성능. GPT-5 Pro는 박사 수준 과학 문제 88.4% 정답률. 이전 모델 대비 사실 오류 최대 80% 감소. [13]

GPT-5.3 Instant (2026.3.3): 웹 검색 결과 연동 환각 26.8% 감소, 불필요한 경고·반복·딱딱한 문체 제거. 출시 1시간 후 GPT-5.4 예고. [14][15]

  • ChatGPT Plus 사용자: GPT-5 Thinking 주당 3,000 메시지 이용 가능
  • 파일 첨부 한 번에 최대 20개로 확대 (기존 10개)
  • 한국어: GPT-5.1부터 존댓말·반말 구분 문제 상당 부분 개선 [C-45]
  • GPT-4o 계열 전면 은퇴 (2026.2.13) — API 접근은 유지 [16]
GPT-5.2 API 가격: 입력 $1.75 / 출력 $14 (100만 토큰 기준) [17]
GPT-5.2 Pro: 입력 $21 / 출력 $168 (100만 토큰 기준)
ChatGPT Plus: 월 $20 유지
주의점

OpenAI는 2025년 8월~2026년 3월 사이에 GPT-5 → 5.1 → 5.2 → 5.3 Instant까지 약 7개월 만에 4번 업데이트를 했습니다. 이 속도로 기업이 특정 모델 버전에 의존하는 시스템을 구축하면 유지보수 비용이 급격히 증가할 수 있습니다. API 버전 고정(pin) 정책을 미리 확인하는 것이 중요합니다.

Nano Banana (Google)

🍌
Nano Banana 2
Google DeepMind · 2026년 2월 26일 출시

Google의 AI 이미지 생성 서비스 브랜드. 인도 등 아시아 시장에서 2025년 8월 처음 출시되어 수백만 장 이미지를 생성하며 바이럴 히트를 기록했습니다. [18][19][20]

버전별 출시 흐름:

  • Nano Banana 1 (2025년 8월): 첫 출시, Gemini 앱에 통합. 특히 인도에서 바이럴

Nano Banana 2 주요 기능 [18]:

  • 해상도 512px ~ 4K, 다양한 화면 비율 지원
  • 캐릭터 최대 5명, 오브젝트 최대 14개 일관성 유지 (스토리보드 제작 가능)

  • Gemini API / Google AI 스튜디오에서 개발자 이용 가능
인사이트

Nano Banana는 Midjourney, DALL-E, Stable Diffusion이 지배하던 AI 이미지 생성 시장에 Google이 본격 진입했음을 알리는 신호입니다. 특히 Google Ads 연동은 "광고 이미지를 AI로 자동 생성 → 즉시 집행"하는 워크플로를 가능하게 해, 마케터 입장에서는 주목할 만합니다. 한국 광고 시장에서의 활용 가능성도 점검이 필요합니다.

Section 11

종합 분석 및 시사점

2025년 말 ~ 2026년 3월: AI 산업 3대 트렌드

1. "에이전트 전쟁" 본격화

단순 대화 AI에서 "자율 실행 에이전트"로 무게 중심 이동. Claude Code 멀티에이전트, Cursor Cloud Agents, Devin 2.2 GUI 조작, Replit Agent 200분 자율 실행, Perplexity Computer 19모델 오케스트레이션 — 모두 같은 방향입니다. 2026년 AI는 "도와주는 도구"에서 "대신 실행하는 팀원"으로 역할이 전환됩니다.

2. 모델 가격의 빠른 하락과 성능 상향 평준화

Claude Sonnet 4.6이 이전 세대 Opus 수준의 성능을 더 낮은 가격에 제공. GPT-5.2 API 입력 단가는 GPT-4o 대비 크게 낮아짐. Gemini Flash 계열은 $0.30~0.50/100만 토큰으로 극도로 저렴. 고성능 AI의 진입 장벽이 낮아지면서, 비용보다 "어떤 에이전트 설계를 하느냐"가 경쟁력의 핵심이 됩니다.

3. 기업 M&A 및 투자 가속

Cognition: Windsurf ~$250M 인수 → $400M 추가 유치 → $10.2B 밸류. Cursor: 시리즈 D $2.3B → $29.3B. GenSpark: 시리즈 B $275M → $1.25B. Replit: $9B 예정. AI 인프라·에이전트 플랫폼에 천문학적 자금이 유입되며, 통합과 도태가 동시에 가속 중.

기회 요소 및 제언

  • 비용 최적화: Claude Sonnet 4.6(Opus급 성능), Gemini Flash($0.30/M) 조합으로 API 비용을 기존 대비 60~80% 절감 가능. 기존 GPT-4o 기반 서비스 전환 검토 시점
  • 코딩팀 생산성: Cursor BugBot Autofix + Cloud Agents 도입 시 PR 리뷰 시간 30~50% 절감 예상. 소규모 개발팀에서 효과 극대화
  • 이미지 광고 자동화: Nano Banana 2 (Google Ads 연동)를 활용한 광고 소재 대량 생성 → A/B 테스트 파이프라인 검토 가치
  • 멀티 에이전트 설계: Claude Code 서브에이전트 + GenSpark + Perplexity Computer 조합으로 반복적 업무(리서치→분석→보고서) 자동화 가능

위험 요소

  • 모델 버전 리스크: OpenAI가 7개월에 4번 업데이트하는 속도 — 특정 버전에 강하게 의존하는 프로덕션 시스템은 유지보수 비용 급증 위험
  • 락인(Lock-in): Cursor·Windsurf·Devin 모두 자체 에코시스템 구축 중. 초기 도입 전에 벤더 전환 비용 사전 평가 필요
  • 가격 변동성: GenSpark·Perplexity Max 등 고가 구독 서비스는 스타트업에게 부담. 무료/저가 플랜의 기능 제한 변경 모니터링 필요
