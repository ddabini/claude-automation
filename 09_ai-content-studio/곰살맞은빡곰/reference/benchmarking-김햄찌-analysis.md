## 4. 콘텐츠 전략 분석

### 제목 패턴
- **4글자 사자성어/신조어**: 조삼모사, 자강두찐, 칠전팔큐, 폼생폼사, 각축지세
- **직장인 키워드**: 직장인이여~, 업무의 일환, 직장인의 언어영역
- **공감형 문장**: 한 번씩은 겪어본 것, 의외로 많은 소비형태
- **짧고 임팩트 있는 2~6글자**가 대부분

### 콘텐츠 주제 분류
1. **직장 생활** — 출퇴근, 회의, 개발자 소통, PT, 야근
2. **일상 공감** — 만성피로, 음식, 소비, 연말
3. **인간관계** — 동료, 점심 메이트, 선후배
4. **자기계발 패러디** — 미라클 모닝, 마음다짐, 주의력

### 업로드 패턴
- Shorts: 주 2~3회
- 풀버전: 주 1~2회 (Shorts 확장판)
- 커뮤니티 게시물: 주 1~2회 (팬과 소통)
- **Shorts + 풀버전 이중 업로드 전략** — 같은 소재를 Short (티저) + 풀버전으로 분할

### 영상 구조
- **풀버전 길이**: 44초 ~ 1분 10초 (1분 내외)
- **자막이 핵심**: 햄스터가 직접 대사를 하는 형태
- **시리즈 묶기**: "현대인들이 자주 겪는 기이현상" 등 시리즈 카테고리
- **워터마크 없음**: 유료 플랜 사용

## 5. 캐릭터 일관성 비결 (추정)

### 왜 김햄찌 햄스터는 항상 같아 보이는가?

1. **Nano Banana로 시작 이미지 생성**
   - 같은 캐릭터 reference를 일관되게 사용
   - 이미지 단계에서 캐릭터 외형을 고정

2. **멀티 도구 양산 후 선별**
   - 같은 시작 이미지를 Sora, Kling, Higglesfield 등에 동시 투입
   - 캐릭터가 가장 일관된 결과물만 선택
   - 한 장면에 최소 3~4회 생성 후 best pick

3. **유료 플랜의 품질 차이**
   - Sora (유료), Kling (유료 추정), Hailuo Premium (유료)
   - 유료 플랜 = 고해상도 + 워터마크 없음 + 더 나은 모션 품질

4. **편집으로 마무리**
   - CapCut에서 여러 클립을 이어붙이기
   - 한 영상 = 여러 도구에서 생성된 클립의 조합
   - 각 장면별로 가장 잘 나온 클립을 선택

### 모션 자연스러움의 비결

- **여러 도구에서 생성 → 자연스러운 것만 선별** (슬로우모션 같은 부자연스러운 결과는 버림)
- **짧은 클립 단위 편집** (6초 클립을 잘라서 필요한 2~3초만 사용)
- **Sora의 모션 품질**이 현재 가장 우수 (Hailuo보다 자연스러움)

## 7. 빡곰에 적용할 인사이트

### 즉시 적용 가능

| 항목 | 김햄찌 방식 | 빡곰 적용 |
|------|-----------|----------|
| 멀티 도구 | Sora+Kling+Higglesfield+Nano Banana | Hailuo Free + Kling + Sora(향후) |
| 양산 후 선별 | 같은 장면 4개 도구로 생성 → best pick | 같은 프롬프트로 3~4회 생성 → 선별 |
| 짧은 클립 편집 | 6초 클립에서 2~3초만 사용 | Hailuo 6초 → 핵심 구간만 추출 |
| 자막 대사 | 햄스터가 직접 말하는 형태 | 빡곰 대사 자막 추가 |
| 효과음 | Capcut + suno | CapCut 효과음 + suno 배경음악 |

### 중기 과제

| 항목 | 설명 |
|------|------|
| TTS 도입 | eleven labs로 빡곰 전용 목소리 만들기 |
| Sora 도입 | 모션 품질이 가장 우수 — 유료 플랜 검토 |
| Higglesfield 테스트 | 김햄찌가 Hailuo 대신 선택한 도구 — 테스트 필요 |
| 시리즈화 | "사회생활 만점 빡곰" 같은 시리즈 카테고리 |
| 풀버전+Shorts 전략 | 1분 풀버전 → Shorts 티저 이중 업로드 |

### 우리의 현재 한계 vs 김햄찌

| 항목 | 김햄찌 | 빡곰 (현재) |
|------|--------|------------|
| 영상 도구 | 유료 4개 (Sora+Kling+Higglesfield+Nano Banana) | 무료 1개 (Hailuo Free) |
| 이미지 도구 | Nano Banana | MCP Gemini + Nano Banana |
| 음향 | Capcut + suno + eleven labs | 없음 (음향 미적용) |
| 편집 | CapCut | 미적용 |
| 워터마크 | 없음 (유료) | 있음 (무료) |
| 양산량 | 장면당 4+ 후보 | 장면당 1~2 후보 |

### 핵심 결론

> **김햄찌의 품질 비결은 단일 도구의 성능이 아니라, 멀티 도구 양산 후 선별 + 편집 + 음향의 3단 프로세스.**
>
> 우리가 당장 따라잡으려면:
> 1. **Hailuo에서 같은 장면 3~4회 생성 후 best pick** (무료로도 가능)
> 2. **CapCut에서 편집 + 효과음 추가** (무료)
> 3. **suno로 배경음악 생성** (무료 tier 있음)
> 4. 이 3단계만 추가해도 현재 대비 품질 대폭 향상
