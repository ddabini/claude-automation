## 제목 패턴 (김햄찌 벤치마킹 적용)

### 고성과 제목 유형

**1. 4글자 사자성어/신조어 (최고 성과)**
- 김햄찌 사례: 조삼모사(336만), 폼생폼사(343만), 칠전팔큐(300만), 각축지세(280만)
- 빡곰 적용 예: `자강두찐`, `곰분투기`, `타의추동`, `명곰답게`
- 핵심: 4글자 한 단어로 영상 전체 내용을 함축. 검색성과 임팩트 동시 확보

**2. 공감형 키워드 (높은 성과)**
- 김햄찌 사례: 선택적 만성피로(441만), 보편적인 기대감(344만)
- 빡곰 적용 예: `합리적 지각`, `계획적 야식`, `본능적 눕기`, `무의식적 새벽배달`
- 핵심: "[형용사] + [공감 행동]" 패턴. 시청자가 자기 이야기로 느끼게

**3. 직장인 타깃 키워드**
- 김햄찌 사례: 직장인이여~(256만), 업무의 일환(250만), 직장인의 언어영역(313만)
- 빡곰 적용 예: `월요병 처방전`, `야근의 품격`, `점심 선택장애`, `회의실 생존법`

**4. 짧은 임팩트형 (2~4글자)**
- 김햄찌 사례: 제철음식(384만), 대박기원(90만)
- 빡곰 적용 예: `퇴근치맥`, `혼밥왕`, `늦잠`, `배달왔다`

### 제목 작성 규칙
- 6글자 이내가 기본 (최대 10글자)
- 설명하지 말고 함축할 것
- 물음표(?) / 느낌표(!) 최소화 — 담담한 톤
- 해시태그는 제목에 넣지 않음 (설명란에만)
- 영어 혼용 금지 (한글만)

## 콘텐츠 주제 카테고리

### A. 직장 생활 (핵심 카테고리)
출퇴근, 회의, 야근, 점심, 동료, 회식, 연차, 월요일

- 출근 지하철 (서있기 / 달려가기)
- 사무실 (졸기 / 모니터 응시 / 커피 마시기)
- 점심 (뭐 먹을지 고민 / 혼밥)
- 회의실 (발표 / 멍때리기)
- 야근 (빈 사무실 / 빈 커피잔)
- 퇴근 (해방감 / 지하철 자리 앉기)

### B. 혼자 일상 (서브 카테고리)
배달음식, TV, 주말, 늦잠, 편의점, 혼술, 치맥

- 집 소파 (TV 보기 / 치맥 / 배달 수령)
- 주방 (라면 끓이기 / 냉장고 앞 서성이기)
- 편의점 (심야 간식 / 컵라면 / 창가 혼밥)
- 주말 (눕기 / 안 나가기 / 늦잠)

### C. 자기계발 패러디 (바이럴 포텐셜)
운동, 다이어트, 미라클모닝, 결심, 작심삼일

- 헬스장 (덤벨 들다 포기 / 러닝머신 걷기 / 스트레칭만)
- 미라클모닝 시도 → 실패
- 새해 다짐 → 작심삼일
- 다이어트 중 배달 주문

### D. 계절/이벤트 (시의성)
명절, 크리스마스, 벚꽃, 여름 더위, 연말

- 연말 회식 시즌
- 크리스마스 혼자 보내기
- 추석/설날 (가족 없는 빡곰의 명절)
- 여름 더위 (북극곰인데 더운 한국)
- 봄 벚꽃 (혼자 벚꽃놀이)

## 업로드 전략

### 주간 업로드 케이던스

| 요일 | 콘텐츠 | 비고 |
|------|--------|------|
| 월 | Shorts 1개 | 출근/월요병 주제 (시의성) |
| 수 | Shorts 1개 | 주중 일상/직장 주제 |
| 금 | Shorts 1개 + 풀버전 1개 | 퇴근/주말 주제. 풀버전은 주간 하이라이트 |

- **주 목표**: Shorts 3개 + 풀버전 1개
- **초기 목표 (현실적)**: Shorts 2개 + 풀버전 0~1개 (제작 역량 구축 기간)

### 이중 업로드 전략 (김햄찌 방식)
1. **풀버전 먼저 제작** (44초 ~ 1분 10초)
2. **풀버전에서 핵심 장면 추출 → Shorts** (15~30초)
3. **동시 업로드**: 풀버전 + Shorts
4. Shorts가 바이럴 → 풀버전으로 유입 유도

### 업로드 시간
- Shorts: **오전 7~8시** (출근 시간대) 또는 **오후 6~7시** (퇴근 시간대)
- 풀버전: **금요일 오후 6시** (주말 시작 시점)

## 자막/대사 스타일

### 기본 원칙
- 빡곰이 직접 말하는 1인칭 (내래이션 아님)
- 짧은 문장 (1~2줄)
- 반말 + 약간 건조한 톤
- 이모티콘/특수문자 최소 사용

### 예시
```
(출근 지하철)
"...또 월요일이다"
"분명 어제까지 주말이었는데"

(배달음식 도착)
"기다렸다"
"...다이어트는 내일부터"

(야근)
"나만 남았다"
"시계를 봤다. 9시 반."
"다시 봤다. 9시 31분."

(편의점 컵라면)
"이 시간엔 이게 맞다"
```

### 자막 스타일
- 폰트: 깔끔한 고딕 (Pretendard 또는 Noto Sans KR)
- 위치: 하단 중앙 (Shorts 세로형에 맞춤)
- 색상: 흰색 텍스트 + 검정 외곽선 또는 반투명 검정 배경
- 크기: 읽기 편한 크기 (화면 너비의 80% 이내)

## 썸네일/커버 전략

- Shorts 썸네일: 자동 생성 (YouTube가 자동 선택) — 초기에는 신경 쓰지 않아도 됨
- 풀버전 썸네일: 빡곰 얼굴 클로즈업 + 짧은 텍스트 (2~4글자)
- 프로필 이미지: 빡곰 정면 얼굴 (현재 사용 중)
