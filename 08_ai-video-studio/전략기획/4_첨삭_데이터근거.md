## 데이터 보완 섹션별 상세

### Why Now — 시장 규모 보강

**현재**: 경쟁사 사례 언급 수준
**보강 필요 내용**:

```
AI 영상 생성 시장 규모:
- 글로벌 AI Video Generation 시장: 2024년 기준 약 $5.1B
  → 2030년 $29.1B 예상 (CAGR 34.2%)
  [확인 필요: Grand View Research 또는 MarketsandMarkets 최신 보고서]

국내 영상 콘텐츠 제작 시장:
- 국내 기업 영상 콘텐츠 제작 외주 평균 비용: 건당 50~300만 원
  [확인 필요: 국내 영상 제작사 단가 조사]
- AI 툴로 직접 제작 시 비용 구조 완전히 달라짐

쇼트폼 영상 폭발적 성장:
- 유튜브 쇼츠 월간 시청자: 2024년 기준 700억 회/일
  [확인 필요: YouTube 공식 발표 최신치]
- 영상 콘텐츠 수요는 증가하나 전문 제작 역량 공급은 부족
```

### MVP 완료 기준 — 성능 수치 근거

**현재**: "90% 이상 자막 성공률" 수치의 근거 없음
**보강 내용**:

```
Whisper API 한국어 성능 벤치마크:
- OpenAI Whisper large-v3: 한국어 WER (Word Error Rate) 약 8~12%
  → 정확도 약 88~92%
  [출처: OpenAI Whisper GitHub 공식 벤치마크]
- 단, 배경 소음 있는 영상은 정확도 70% 이하로 하락 가능
- MVP 권장 조건: 음성이 명확한 영상 기준 (배경음악 없거나 낮은 경우)

Runway Gen-3 Alpha 생성 시간:
- 5초 영상 기준: 약 40~90초 소요 (서버 부하에 따라 변동)
  [확인 필요: 실제 API 호출 시간 측정치]
- UX 임계값: 3분 이상 대기 시 이탈률 급증 (업계 일반 기준)
  → 생성 진행률 바 + 예상 시간 표시 필수
```

  [출처: Stack Overflow Developer Survey 2024]
