# /strategy — 브랜드 전략 완전 자동화

사용자가 `/strategy`를 입력하면 브랜드 전략 전 과정을 자동으로 실행한다.
입력 최소화, 산출물 최대화. **레퍼런스 퍼스트 원칙에 따라 레퍼런스 수집을 먼저 진행한다.**

---

## 실행 순서

### Step 1. 브리프 수집
4가지만 물어본다:
```
1. 브랜드명 (또는 가칭)
2. 무엇을 파는 브랜드인가 (업종·서비스·상품)
3. 어떤 사람에게 팔고 싶은가 (타깃)
4. 참고하고 싶은 브랜드 또는 레퍼런스 URL (없으면 생략 가능)
```
→ 입력 즉시 자동 실행 시작.

### Step 2. 리서치 + 레퍼런스 수집 (자동, 병렬)
두 에이전트를 **동시에** 호출합니다:

**`research-crawler`** (팩트 데이터):
- 시장 규모·성장률·주요 트렌드 조사
- 경쟁사 3~5개 분석 (서비스·포지셔닝·강점/약점)
- 타깃 고객 행동 패턴·니즈 데이터
- **출처 검증된 조사 보고서 제출**

**`design-reference-archiver`** (비주얼 레퍼런스):
- 시드 URL이 있으면 → 해당 브랜드/사이트 분석 + 유사 브랜딩 레퍼런스 확장 수집
- 시드 URL이 없으면 → 업종·타깃 키워드 기반 브랜딩 레퍼런스 수집
- 경쟁사 비주얼 아이덴티티, 톤앤매너 사례 수집
- **디자인/구성 포인트 보고서 제출**

### Step 3. 팀장 컨펌
두 보고서(조사 보고서 + 디자인 포인트 보고서)를 사용자에게 통합 보고:
- 팩트 데이터 방향이 전략 의도와 맞는지 확인
- 비주얼 레퍼런스 방향이 브랜드 의도와 맞는지 확인
- **승인** → 전략 수립 시작 (두 보고서 데이터를 전략에 반영)
- **수정 지시** → 해당 에이전트가 보완 후 재제출

### Step 4. 브랜드 전략 5종 자동 생성

레퍼런스 분석 결과를 반영하여 아래 순서대로 멈추지 않고 연속 생성:

**① 시장·경쟁 분석**
- 업종 트렌드, 주요 경쟁사 3~5개 분석
- 경쟁사 포지셔닝 맵
- 차별화 기회 포인트
- (레퍼런스에서 발견한 경쟁사 전략 인사이트 포함)

**② 브랜드 포지셔닝**
- 브랜드 본질 (Brand Essence) 한 문장
- 포지셔닝 스테이트먼트
- 핵심 가치 3가지
- 브랜드 퍼스널리티 (형용사 5개)

**③ 타깃 페르소나**
- 메인 페르소나 1명 (이름·나이·직업·라이프스타일·페인포인트·욕구)
- 서브 페르소나 1명
- 타깃이 브랜드를 선택하는 핵심 이유

**④ 브랜드 네이밍 & 슬로건**
- 네이밍 후보 5개 (각각 의미·발음·기억 용이성 설명)
- 슬로건 후보 3개
- 추천안 + 이유

**⑤ 브랜드 커뮤니케이션 전략**
- 핵심 메시지 계층 (메인·서브·증거)
- 채널별 톤앤매너 (SNS·오프라인·광고)
- 콘텐츠 방향 3가지
- (레퍼런스에서 추출한 톤앤매너·비주얼 방향 반영)

### Step 5. 통합 전략 요약 (1페이지)
위 5종을 한 페이지로 압축한 **브랜드 전략 요약서** 자동 생성.
의사결정자가 5분 안에 전체 방향을 파악할 수 있게.

### Step 6. 저장 및 보고
```
✅ 브랜드 전략 완료

📁 01_baroezip/[브랜드명]/전략기획/ (바로이집) 또는 클로드/[브랜드명]/전략기획/ (기타)
  ├── 1_시장경쟁분석.md
  ├── 2_브랜드포지셔닝.md
  ├── 3_타깃페르소나.md
  ├── 4_네이밍슬로건.md
  ├── 5_커뮤니케이션전략.md
  └── 0_전략요약서.md   ← 1페이지 핵심 요약

📁 [프로젝트명]/레퍼런스/
  ├── README.md          ← 포인트 보고서
  └── links.md           ← 수집 링크 모음

다음 단계 제안:
→ 디자인 파이프라인으로 브랜드 아이덴티티 디자인 시작
  (에셋 에이전트가 메인폰트 선정 + 에셋 생성 → Figma 캡처)
→ /planning 으로 사업 기획서 작성
```

---

## 품질 기준
- 모든 주장에 근거 또는 데이터 포함
- 추상적 표현 금지 — 구체적 수치·사례·키워드로 표현
- 네이밍은 실제 도메인·상표 등록 가능성 언급
- 슬로건은 한국어·영어 각각 제시
- 레퍼런스 분석 인사이트가 전략 문서 곳곳에 반영되어 있을 것
