# Strategy Planning Agent Memory

## 사용자 브랜드 컨텍스트

## 전략 작성 패턴

- 사용자는 McKinsey 스타일 프레임워크 선호
- 브랜드 전략 → 디자인 → 개발 순서의 워크플로우
- 기획서는 실행 가능한 구체적 액션 아이템 포함 필수

## 주의사항

- 전략 문서 작성 시 항상 한국어 사용
- 과도한 전문용어 지양 → 실무진도 바로 이해할 수 있는 수준

## 관련 에이전트 메모리 참조

| 에이전트 | 참조 시점 | 참조 내용 |

| design-reference-archiver | 기획 완료 후 전달 시 | 기획 브리프의 컨셉·톤앤매너를 레퍼런스 검색 키워드로 변환 |
