# 에이전트 공유 메모리 (Shared Agent Memory)

이 폴더는 에이전트 간 실행 결과를 공유하는 임시 메모리입니다.
각 에이전트가 작업 완료 시 요약을 여기에 기록하고, 다음 에이전트가 참조합니다.

---

## 사용 규칙

### 기록 시점
- 에이전트가 **다른 에이전트에게 넘길 결과물**이 있을 때 기록
- 파이프라인 실행 중 단계 간 연계 정보 전달용

### 기록 형식
```
## [에이전트명] → [대상 에이전트명] (YYYY-MM-DD)
- 작업: [수행한 작업 한 줄 요약]
- 산출물: [파일 경로 또는 결과 요약]
- 다음 단계 참고: [다음 에이전트가 알아야 할 핵심 정보]
```

### 정리 규칙
- `session-closer`가 세션 종료 시 공유 메모리를 정리
- 학습 포인트는 관련 에이전트 개별 메모리로 흡수
- 세션 로그(작업 전달 기록)는 삭제
- 이 MEMORY.md 파일의 사용 규칙 섹션은 항상 유지
