# 브랜드 전략 수립 프로세스

---

## 전체 흐름 요약

```
1. 현황 분석 → 2. 브랜드 정의 → 3. 전략 수립 → 4. 기획안 작성 → 5. 실행 및 관리
```

---

## 1단계. 현황 분석 (Discovery)

### 목적
브랜드가 처한 내·외부 환경을 객관적으로 파악하여 전략의 근거를 마련한다.

### 주요 업무

#### 1-1. 내부 분석
- 비즈니스 현황 파악 (매출, 고객 수, 채널, 주요 상품·서비스)
- 기존 브랜드 자산 점검 (로고, 슬로건, 톤앤매너, 인지도)
- 조직 내부 인식 조사 (경영진 인터뷰, 구성원 설문)
- 강점·약점 정리

#### 1-2. 외부 분석
- 시장 규모 및 성장 트렌드 조사
- 경쟁사 브랜드 분석 (포지셔닝, 메시지, 채널 전략)
- 고객 분석 (타겟 페르소나, 구매 여정, Pain Point)
- 사회·문화·기술 트렌드 스캔 (PEST 분석)

#### 1-3. 종합 정리
- SWOT 분석
- 브랜드 갭(Gap) 분석: 현재 인식 vs 목표 인식

### 산출물
- 시장·경쟁 분석 보고서
- 고객 페르소나 카드
- SWOT 요약표

---

## 2단계. 브랜드 정의 (Brand Definition)

### 목적
브랜드의 본질적인 정체성과 약속을 명확하게 언어화한다.

### 주요 업무

#### 2-1. 브랜드 코어 설정
| 요소 | 설명 | 예시 |
|------|------|------|
| **미션(Mission)** | 브랜드가 존재하는 이유 | "우리는 왜 이 일을 하는가?" |
| **비전(Vision)** | 장기적으로 달성하고 싶은 모습 | "10년 후 우리 브랜드는?" |
| **핵심 가치(Values)** | 브랜드가 지키는 신념과 원칙 | 3~5개 키워드로 정의 |

#### 2-2. 브랜드 포지셔닝
- 포지셔닝 맵 작성 (X축/Y축 경쟁 요소 설정)
- 핵심 차별점(USP, Unique Selling Point) 도출
- 포지셔닝 스테이트먼트 작성
  > "우리 브랜드는 [타겟]에게 [혜택]을 제공하는 [카테고리]이며, [경쟁 브랜드]와 달리 [차별점]을 가진다."

#### 2-3. 브랜드 퍼스널리티
- 브랜드를 사람으로 표현한다면? (형용사 5~7개)
- 톤앤매너 정의: 말하는 방식, 감정적 느낌
- 브랜드 아키타입 선택 (영웅, 탐험가, 현자, 돌봄이 등)

#### 2-4. 브랜드 프로미스
- 고객에게 일관되게 전달할 핵심 약속 한 문장

### 산출물
- 브랜드 아이덴티티 정의서
- 포지셔닝 맵
- 브랜드 퍼스널리티 가이드

---

## 3단계. 전략 수립 (Strategy)

### 목적
정의된 브랜드 정체성을 실제 시장에서 구현할 전략 방향을 설계한다.

### 주요 업무

#### 3-1. 타겟 전략
- 핵심 타겟 / 잠재 타겟 구분
- 타겟별 브랜드 접점(Touchpoint) 정의
- 채널별 타겟 행동 패턴 분석

#### 3-2. 메시지 전략
- 핵심 메시지 (Core Message) 도출
- 타겟별 메시지 변형 (동일한 가치를 다르게 표현)
- 키 비주얼·슬로건 방향 설정

#### 3-3. 채널 전략
- 소유 채널 (웹사이트, SNS, 뉴스레터 등)
- 유료 채널 (광고, 협찬, 인플루언서)
- 획득 채널 (PR, 바이럴, 입소문)
- 채널별 역할·목표 KPI 설정

#### 3-4. 브랜드 경험 전략
- 고객 여정 지도(Customer Journey Map) 작성
- 핵심 브랜드 접점 우선순위 선정
- 오프라인 경험 / 온라인 경험 설계 방향

#### 3-5. 중장기 로드맵
- 단기(3~6개월): 브랜드 인지도 구축
- 중기(6~18개월): 신뢰도·선호도 강화
- 장기(18개월~): 충성 고객 확보, 브랜드 자산 축적

### 산출물
- 브랜드 전략 요약서 (1-pager)
- 타겟·채널·메시지 전략 문서
- 브랜드 로드맵

## 4단계. 기획안 작성 (Planning)

### 목적
전략을 실행 가능한 구체적인 기획안으로 문서화하여 의사결정·공유에 활용한다.

### 기획안 구성 (표준 목차)

```
1. Executive Summary (핵심 요약, 1장)
2. 현황 및 문제 인식
   - 시장 환경
   - 고객 인식 현황
   - 해결해야 할 핵심 과제
3. 브랜드 전략 방향
   - 미션·비전·가치
   - 포지셔닝
   - 핵심 타겟
4. 실행 계획
   - 메시지·크리에이티브 방향
   - 채널별 실행 전술
   - 타임라인 (간트차트)
5. 예산 계획
6. KPI 및 성과 측정 방법
7. 리스크 및 대응 방안
```

### 기획안 작성 원칙
- **Why → What → How** 순서로 논리를 전개
- 데이터와 인사이트를 근거로 주장 뒷받침
- 경영진 보고용은 1-pager 요약본 병행 작성
- 시각화 우선: 텍스트보다 도표·그래프·다이어그램 활용

### 산출물
- 브랜드 전략 기획안 (PPT / PDF)
- 실행 계획서 (Action Plan)
- 예산 계획서

---

## 5단계. 실행 및 관리 (Execution & Management)

### 목적
전략이 현장에서 일관되게 구현되고, 지속적으로 개선될 수 있도록 관리한다.

### 주요 업무

#### 5-1. 브랜드 가이드라인 배포
- 내부 구성원 공유 및 교육
- 외부 협력사 브리핑

#### 5-2. 실행 모니터링
- 채널별 KPI 주기적 점검 (주간 / 월간)
- 고객 반응·피드백 수집 및 분석

#### 5-3. 브랜드 감사(Brand Audit)
- 분기 또는 반기별 브랜드 인식 조사
- 전략과 실행 간 정합성 점검
- 필요시 전략 수정(Pivoting)

### KPI 예시
| 영역 | 지표 |
|------|------|
| 인지도 | 브랜드 검색량, 소셜 언급량, 도달 수 |
| 선호도 | 브랜드 호감도 조사, NPS |
| 행동 | 웹사이트 방문, 전환율, 재구매율 |
| 충성도 | 구독자 수, 앱 재방문율, 추천 비율 |

## 업무 협업 방식

### 역할 분담 기본 구조
| 역할 | 주요 책임 |
|------|----------|
| 전략기획 리드 | 전체 방향 설정, 기획안 총괄, 의사결정 |
| 리서치 | 시장·경쟁·고객 조사 및 인사이트 도출 |
| 크리에이티브 | 메시지·비주얼 개발 (디자인·카피 협업) |
| 마케팅 실행 | 채널별 콘텐츠 기획·운영 |
| 데이터 분석 | KPI 측정, 성과 보고 |

### 회의 운영 방식
- **킥오프 미팅**: 목표·범위·일정 공유
- **주간 스탠드업**: 진행 상황 점검 (15~30분)
- **월간 리뷰**: KPI 기반 성과 점검 및 방향 조정
- **분기 전략 회의**: 전략 유효성 검토 및 수정

### 문서 관리 원칙
- 모든 기획안은 버전 관리 (v1.0, v1.1...)
- 최종본과 작업본 폴더 분리 보관
- 의사결정 이력 별도 기록

## 클로드 에이전트 활용 포인트

| 작업 | 활용 방법 |
|------|----------|
| 시장·경쟁 분석 | 정보 수집 후 분석 요청, 프레임워크 적용 |
| 페르소나 작성 | 타겟 조건 제공 → 페르소나 초안 생성 |
| 포지셔닝 | 경쟁사 정보 입력 → 차별점 도출 |
| 기획안 초안 | 목차 구조·핵심 메시지 초안 작성 |
| 슬로건·카피 | 브랜드 방향 공유 → 다수 시안 생성 |
| 보고서 요약 | 긴 문서 → 핵심 인사이트 요약 |
