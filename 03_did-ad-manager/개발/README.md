# DID 광고 프로그램

4K 디스플레이용 광고 슬라이드쇼 + 웹 기반 관리자 페이지

---

## 파일 구성

```
개발/파일/
├── display.html   ← 디스플레이 화면 (이걸 4K 모니터 전체화면으로 열기)
└── admin.html     ← 관리자 페이지 (소재 업로드/관리)
```

---

## 빠른 시작 (5분)

### 1단계: admin.html 열기
- `admin.html` 파일을 브라우저에서 열기 (더블클릭)
- 소재 업로드 영역에 이미지/영상 파일 드래그앤드롭

### 2단계: 설정
- 왼쪽 메뉴 "설정" 클릭
- 기본 노출 시간, 전환 효과 설정 후 "설정 저장"

### 3단계: 디스플레이 열기
- 상단 오른쪽 "▶ 디스플레이 열기" 버튼 클릭
- `display.html`이 새 탭으로 열림
- F 키를 눌러 전체화면으로 전환

---

## 사용 방법

### 소재 업로드
- 지원 형식: JPG, PNG, GIF, WebP, MP4, WebM
- 여러 파일 동시 업로드 가능
- 10MB 초과 파일은 경고 메시지 표시 (업로드는 가능)

### 소재 관리
| 기능 | 방법 |
|------|------|
| 순서 변경 | 카드의 ▲▼ 버튼 클릭 |
| 노출/숨김 | 카드의 눈 아이콘 버튼 클릭 |
| 개별 노출시간 | 카드의 "노출시간" 입력란에 초 입력 |
| 삭제 | 카드의 🗑 버튼 클릭 (확인 후 삭제) |

### 키보드 단축키 (display.html)
| 키 | 동작 |
|---|------|
| F | 전체화면 전환 |
| Space | 재생 / 일시정지 |
| → | 다음 소재 |
| ← | 이전 소재 |

---

## 저장공간 안내

- 브라우저 localStorage: 약 5~10MB
- 고해상도 이미지/영상 업로드 시 용량 부족 가능
- 용량 부족 시: 화면 상단 "저장공간" 표시가 빨간색으로 변함
- 해결: 오래된 소재 삭제 또는 "백업/복원" 탭에서 데이터 내보내기 후 재업로드

---

## 다른 컴퓨터에서 사용하려면

1. admin.html 열기 → "백업/복원" 탭 → "전체 데이터 내보내기"
2. JSON 파일을 다른 컴퓨터로 복사
3. 다른 컴퓨터에서 admin.html 열기 → "파일 불러오기"로 복원
