🔒 보안 규칙 참고

Firebase 콘솔에서 아래 규칙이 적용되어 있어야 합니다.
주의: 이 규칙은 내부 사용 목적입니다. 외부 공개 시 인증(Auth) 규칙 적용을 권장합니다.

Realtime Database 규칙 (빌드 → Realtime Database → 규칙 탭)
{ "rules": { ".read": true, ".write": true } }
Storage 규칙 (빌드 → Storage → Rules 탭)
rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /{allPaths=**} { allow read, write: if true; } } }
