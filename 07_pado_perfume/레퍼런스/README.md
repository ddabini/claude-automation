```
메인 컬러:
- 딥 오션 블루  #1B4965  (제주 심해 / Byredo 감성)
- 세이지 베이지  #C9B99A  (제주 모래 / 코스탈 럭셔리)
- 크리스탈 화이트  #F7F5F2  (현무암 위 파도 거품 / 프리미엄 공간)

보조 컬러:
- 미드나이트 네이비  #0D2137  (딥 오션 새벽 / 다크 버전 배경)
- 터쿼이즈 포인트  #5FA8D3  (제주 히협재 해변 / 강조 컬러)
- 현무암 블랙  #2D2D2D  (제주 현무암 / 텍스트 컬러)
```

**컬러 방향 근거**:
- 레퍼런스들에서 "화이트/크림 + 딥 블루"가 코스탈 럭셔리의 공통 언어로 확인됨
- Byredo/Tamburins: 블랙/화이트 베이스 (너무 어두울 수 있어 PADO는 블루로 치환)
- Sea Breeze 브랜딩: "아이리 타이포 + 소프트 블루" — PADO 방향과 가장 일치
- 탈색된 자연스러운 톤: 채도를 20-30% 낮춘 "바다에 오래 있다 색이 빠진" 느낌

### 2. 레이아웃 — 세로형 1080×1350 패턴

수집된 레퍼런스에서 발견된 카드뉴스 레이아웃 공통 패턴:

**카드 1장 (브랜드 임팩트) — 추천 패턴**:
```
[여백 상단 20%]
[브랜드 로고 or 영문 헤드라인 — 화면의 40% 비중]
[서브 카피 1줄 — 화면의 15% 비중]
[이미지/텍스처 배경 — 화면 전체 가득]
[여백 하단 15%]
```
- 텍스트가 "떠 있는" 느낌 (배경이 텍스트를 밀지 않음)
- Byredo 스타일: 헤드라인 1개만, 나머지는 이미지가 말함

**카드 2장 (제품 스토리) — 추천 패턴**:
```
[제품 사진 풀블리드 또는 60% 비중]
[스토리 텍스트 블록 — 세리프 소문자, 상단 또는 하단 배치]
[향수 노트 키워드 3-4개 — 작은 사이즈, 줄 간격 넓게]
```
- "제품 + 시" 조합: 이미지가 주, 텍스트가 주석
- Ffern 스타일: 원재료(제주 동백/해염/해초) 클로즈업 사진이 스토리

**카드 3장 (라인업+CTA) — 추천 패턴**:
```
[라인업 제품명 목록 — 세로 배열, 균등 간격]
[CTA 텍스트 — 가장 작은 사이즈로 하단 배치]
[화이트 또는 딥 오션 블루 단색 배경]
```
- 제품을 많이 보여주되 과하지 않게 — 여백이 곧 프리미엄

### 3. 타이포그래피 — 폰트 방향

**발견된 경향**:
- 니치 퍼퓸 브랜드 95%: 세리프 계열 타이포 사용 (산세리프는 보조)
- 헤리티지/클래식 세리프 > 모던 세리프 > 지오메트릭 산세리프 순서
- 대문자 또는 소문자 통일 (대소문자 혼합 지양)
- 자간(letter-spacing): 일반 대비 1.5-3배 넓게 (프리미엄 여백 표현)

**PADO 추천 폰트 시스템**:
```
헤드라인(영문): Cormorant Garamond or EB Garamond
  → 헤리티지 감성, 바다의 우아함, 여백 활용 최적
  → Byredo 커스텀 폰트와 같은 "modernist principles" 계열

본문(영문): Futura PT Light or Gill Sans Light
  → 헤드라인 세리프와 대비되는 클린한 서포트 역할

한글 헤드라인: Noto Serif KR or KoPubWorld Batang
  → 세리프 계열로 영문과 톤 통일

한글 본문: Pretendard Light or Regular
  → 가독성 + 모던함 (Tamburins 한국 감성)

자간: 3-5px 넓게 / 줄간격: 1.8-2.0 배수
```

### 4. 이미지/그래픽 — 비주얼 방향

수집된 레퍼런스 공통 이미지 활용 방식:

**권장**:
- 자연광 제품 사진 (스튜디오 조명보다 창가/야외 자연광)
- 텍스처 클로즈업: 파도 물결, 현무암 표면, 모래 결, 동백꽃 잎
- 제품 병 + 자연 배경 (병이 배경에 용해되는 느낌)
- 오버익스포즈(살짝 날린) 라이팅 — 탈색된 톤과 일치
- 블러 처리: 배경 일부를 보케 처리하여 몽환적 느낌

**비권장** (수집 과정에서 걸러낸 패턴):
- AI 생성 이미지 특유의 과도한 그래디언트
- 블롭(blob) 장식 요소
- 기본 파란 버튼 + 강렬한 형광 컬러 조합
- 반짝이(글리터) 이펙트 — 니치 퍼퓸과 어울리지 않음

## 에셋 제작 제안 (다음 단계)

레퍼런스 분석을 바탕으로 design-asset-generator에 전달할 에셋 방향:

### 1장 (브랜드 임팩트)
- 배경: 딥 오션 블루(#1B4965) 풀블리드 or 제주 바다 사진 (오버익스포즈)
- 메인 요소: "PADO" 영문 로고 — Cormorant Garamond, 자간 넓게, 화이트
- 서브 카피: "제주 바다의 기억" — Pretendard Light, 소문자, 상당한 여백
- 레이아웃: 수직 중앙 정렬, 화면의 30% 텍스트 / 70% 공기(여백+배경)

### 2장 (제품 스토리)
- 배경: 크리스탈 화이트(#F7F5F2) or 현무암 텍스처 미묘하게 오버레이
- 제품 사진: 향수 병 자연광 클로즈업 + 제주 현무암 or 해변 소품 (보케 처리)
- 스토리 카피: 3-5줄 세리프 서체, 자간 넓게, 하단 1/3 배치
- 향수 노트: 소문자 영문 키워드 3개 (예: sea salt · jeju moss · amber)

### 3장 (라인업+CTA)
- 배경: 크리스탈 화이트(#F7F5F2) or 세이지 베이지(#C9B99A)
- 제품 라인 3-4개 세로 목록: 영문 이름 + 한글 설명 + 가격
- CTA: "지금 경험하기" — Pretendard Regular, 가장 작은 사이즈
- 브랜드 URL/태그: 하단 여백 안에 배치
