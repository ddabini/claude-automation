## 1. 폰트 구성

### 확정 폰트 시스템

| 역할 | 폰트명 | 굵기 | 용도 |
|------|--------|------|------|
| **메인폰트 (영문 헤드라인)** | Raleway | 100~300 (ExtraLight~Light) | PADO, 브랜드명, 영문 제목 |
| **메인폰트 (한글 헤드라인)** | Paperlogy ExtraBold | 800 | 한글 메인 카피 |
| **서브폰트 (한글/영문 본문)** | Pretendard | 300~400 (Light~Regular) | 스토리 카피, 노트 설명 |
| **포인트폰트 (영문 라인업)** | Josefin Sans | 100~300 | 제품명, 서브 영문 라벨 |

### CDN 링크

```html
<!-- Raleway: 영문 헤드라인 (가벼운 산세리프) -->
<link href="https://fonts.googleapis.com/css2?family=Raleway:wght@100;200;300;400&display=swap" rel="stylesheet">

<!-- Josefin Sans: 영문 제품명 (기하학적 산세리프) -->
<link href="https://fonts.googleapis.com/css2?family=Josefin+Sans:wght@100;200;300&display=swap" rel="stylesheet">

<!-- Pretendard: 한글/영문 본문 -->
<link href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css" rel="stylesheet">

<!-- Paperlogy ExtraBold: 한글 헤드라인 -->
<style>
@font-face {
  font-family: 'Paperlogy';
  src: url('https://cdn.jsdelivr.net/gh/e-hwan/paperlogy@latest/fonts/Paperlogy-8ExtraBold.woff2') format('woff2');
  font-weight: 800;
  font-style: normal;
  font-display: swap;
}
</style>
```

## 5. 렌더링 주의사항 (design-figma에 전달)

### 1. 폰트 fallback 처리
```css
/* Paperlogy 로드 실패 시 fallback */
.headline-ko {
  font-family: 'Paperlogy', 'SUIT', 'Noto Sans KR', sans-serif;
}
```

### 2. SVG 배경 렌더링
- SVG 배경은 `<img>` 태그 또는 CSS `background-image: url(...)` 으로 사용
- `background-size: cover` + `background-position: center` 필수

### 3. 보틀 이미지 합성
- 보틀 SVG를 배경 위에 올릴 때: `mix-blend-mode: multiply` (화이트 배경 위) 또는 `normal` (다크 배경 위)

### 4. 텍스처 오버레이 적용
```css
.texture-overlay {
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background-image: url('assets/backgrounds/texture-wave-overlay.svg');
  opacity: 0.12;
  mix-blend-mode: overlay;
  pointer-events: none;
}
```

### 5. 카드 크기 (HTML 렌더링)
- HTML에서 카드 컨테이너: `width: 1080px; height: 1350px`
- Pikbox 업로드 전 `devicePixelRatio: 2` 캡처 권장 (레티나 해상도)
